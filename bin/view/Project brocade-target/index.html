Brocade FC HBA - Target

Project

Target mode driver for Brocade FC HBAs (BFA_TM)

Endorsing Communities

  • HPC Developer
  • Storage

Introduction

This project would deliver target mode driver for Brocade Fibre Channel HBAs. This target mode driver is designed to work with Solaris COMSTAR software framework which enables a Solaris host to be used as a SCSI Target platform.

Along with this project, we would also like to actively participate and contribute to COMSTAR project.
