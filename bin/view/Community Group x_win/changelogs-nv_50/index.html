X Change Logs: Nevada Builds 50-59

These changes went into the X Consolidation in Solaris Nevada Builds 50-59:

Build 50

  • 6467290 snv_47, fr_CA: xserver doesn't start if French-Canadian keyboard layout is selected
  • 6465198 xdpyinfo shows incorrect extensions list on snv11_46
  • 6471602 Xorg modularization: xbitmaps 1.0.1
  • 6472724 Upgrade xscreensaver to 5.0.1
  • 6426302 [tjds] capturing screen using sdtimage crashes Xserver
  • 6432709 [TX,atok] atok pallet disappear on non admin label when moving the position
  • 6432729 TrustedExtensionsPolicy file should be updated for Japanese Input method
  • 6435331 Trusted X server libXtsol.so does not exist in /usr/lib/amd64 on Trust System
  • 6459852 [tjds] gnome-panel-screenshot cores - hangs desktop
  • 6423858 [Xorg bug 5898] file creation race condition in Xsession

Build 51

  • 6475968 Desegregation of X11 External Libraries [PSARC/2006/557]
  • 6476476 Xorg modularization: libXfixes

  • 6477006 keycode 22 reports keysym 0/NoSymbol on Sun Type 6 Unix keyboard
  • 6477136 Application crashed in FcHideFont if non-alphabetical name exists
  • 6477401 Xorg modularization: SUNWxorg-client\*
  • 6478492 XCloseDisplay() dumps core if libumem is used with UMEM_DEBUG=default
  • 6473283 Xorg Radeon Driver should not emit messages "RADEONSaveScreen" "RADEONDisplayPowerManagementSet"

  • 6428029 Monospace fonts are not displaying Thai words properly.

  • 6476205 [tjds] magnified porttion of screen corrupt when running magnifier

  • 6418052 Direct Rendering Infrastructure (DRI) should be supported by Xorg

Build 53

  • 6409196 freetype2  ftconfig.h defines SIZEOF_LONG to be 8 on 32bit x86
  • 6254329 many fonts display poorly
  • 6488392 PSARC 2006/609: Xserver provider for DTrace
  • 6420311 auto-config improve: Ferrari 4000 starts with blank screen without a config file
  • 6450687 auto-config: radeon - Modes need to be sorted
  • 6482283 radeon_drv.so on dual-headed machines: trouble with phys. resolution, and flicker
  • 6484245 auto-config: radeon: misc fixes to radeon driver
  • 6484688 auto-config: i810: "680x480" on display when monitor is not properly probed
  • 6418254 Solaris needs Nvidia video driver for performance gain

Build 54

  • 6390385 xscreensaver hack anemone kills Xsun when run from commandline in JDS3 on m64 and pgx32
  • 6490500 X consolidation needs to stop building & delivering X server keytables
  • 6395649 at-spi-registryd starts when screen is locked even when accessible device support is off
  • 6461887 GNOME screen lock does not prevent access to other applications via 'alt-tab'

  • 6492147 some packages upgrade fails with zones to Nevada
  • 6492148 package SUNWxwsvr upgrade fails with zones to Nevada
