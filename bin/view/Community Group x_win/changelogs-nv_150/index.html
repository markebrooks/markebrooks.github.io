Nevada Builds 150-159

X Change Logs: Nevada Builds 150-159

These changes went into the X Consolidation in Solaris Nevada Builds 150-159:

Build 150

  • 6988507 X commands have invalid symbol table entries; xterm cannot start 

Build 151

No changes delivered.

Build 152

  • 6516586 DMX support for Solaris x86 and SPARC [PSARC 2010/176]
  • 6935429 xorg.conf.vesa should be delivered with X server [PSARC/2010/161]
  • 6982811 64 bit Xvnc server and Tiger Vncviewer support
  • 6994346 xnv-gate should be built with Studio 12 Update 1 version of CBE
  • 6994566 download-tarballs doesn't work in a fresh clone

Build 153

  • 6996111 validate_pkg failures introduced by VNC & DMX putbacks in snv_152
  • 6996309 Introduce libXcb in Solaris [fix packaging]
  • 6667057 Introduce libXcb in Solaris [PSARC 2010/109]
  • 6907134 Upgrade synergy to 1.3.4
  • 6991716 Removal of 32-bit-only Xorg video drivers
  • 6998036 Update to xcb versions of xlstoms, xlsclients, and xwininfo 

Build 154

  • 6997986 TrueType font can crash libfreetype applications
  • 6992282 Mismerge of ast ddx driver
  • 7001277 Update SourceForge & X.Org URLs for source downloads
  • 7001988 X11R7.6 Update: Font modules

Build 155

  • 7002412 X builds using build 153 pkg bits will fail to generate automatic dependencies
  • 7002660 Remove workaround for 6785384 from X build
  • 7004003 snv_153 breaks remote display (xhosting) of applications
     

Build 156

  • 6943799 Remove references to sun logo in xlock man page
  • 7005752 XFER/NFER/HKTG symbols should be available in XKB keycode table - X server Part
  • 7006573 X11R7.6 Update: Driver modules (except intel)

Build 157

  • 7009421 X pkg build loses tag version information after the first period
  • 7009232 X11R7.6 Update: Documentation modules

Build 158

  • 7009815 Upgrade mesa-demos to 8.0.1
  • 7010050 X consolidation package copyrights need to be updated to 2011
  • 7010818 make download in the X gate should not leave empty files when it fails
  • 7010621 X11R7.6 Update: Protocol modules
  • 7012367 Upgrade xterm from 261 to 267
  • 7012962 Update libX11 to 1.4.1

Build 159

  • 6999968 pkg manifests for xserver lists "preserve" for rbac files
