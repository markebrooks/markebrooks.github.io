Nevada Builds 70-79

X Change Logs: Nevada Builds 70-79

These changes went into the X Consolidation in Solaris Nevada Builds 70-79:

Build 70

  • 6572695 /usr/share/doc/fontconfig/fontconfig-user.html file is missing
  • 6579436 xf86-video-nv upgrade from 2.0.2 to 2.1.2
  • 6569508 Mesa 3D driver for Intel 965
  • 6577118 Xorg drops its gid from 0 to login user's gid
  • 6577124 Xorg gets killed when a non-root user runs 3D apps
  • 6579084 Increasing multi-texture upper limit in Mesa
  • 6551484 Deadlock in libXi if called from java/GTK application
  • 6581910 SUNWxwxft zones upgrade failure if you upgrade from S10u1 to snv_68

Build 71

  • 6582224 Move libXfont to SUNWxwfsw
  • 6509763 \*bdftopcf\* dumps core on stdin fuzz testing
  • 6582231 Xorg modularization: bdftopcf
  • 6543140 Xsun headers need to be delivered in order to build Xsun DDX modules
  • 6582977 X Damage Extension Version 1.1 [LSARC/2007/421]
  • 6583151 SUNWxorg\* packages dependency issue in snv_69.
  • 6573802 xft.hint.patch is using incorrect include path
  • 6583543 Can't build X with JDS CBE 1.6
  • 6585549 X Resize, Rotate and Reflect (RandR) 1.2 [PSARC/2007/430]
  • 6585554 make clean doesn't clean out generated font files
  • 6566519 Open and Front keys stop working on one monitor only
  • 6583707 Radeon driver: CRT2 gets wrong monitor info
  • 6580970 Xorg Server 1.3 [PSARC/2007/434]
  • 6591797 NVIDIA driver needs better support for compiz
  • 6406668 nVIDIA keyboard and screen freeze
  • 6499659 unloading the nvidia driver can result in panic
  • 6578011 "Bad string" error when running nvidia-SunOS-bug-report.sh
  • 6578012 nvidia-SunOS-bug-report.sh should dump smbios info
  • 6578251 nvidia-xconfig should generate mode nvidia-auto-select by default 

Build 72

  • 6552230 Backing store delay between Solaris 9 and Solaris 10
  • 6540571 move auditwrite into xserver
  • 6554807 Trusted Extensions: In XINERAMA windows can't be moved to the lower head (which remains black)
  • 6575597 "property _SCREENSAVER_STATUS" should be included as a default in the trusted config file
  • 6583888 [tcde] possible to view labeled windows immediately after login
  • 6584054 Cannot build TX X server module
  • 6589553 overflow vulnerability in freetype [CVE-2007-2754]
  • 6582847 XErrorDB path is incorrect
  • 6580728 Upgrade Xorg Intel video driver to xf86-video-intel-2.1.0
  • 6586407 CAS issues with Xserver fonts packages with heterogeneous diskless client setup in snv_69/s10u4.

Build 73

  • 6576355 SUNWxwxft has broken links in snv nightly build for Xft
  • 6580918 Bad quality of bold font emulation in Xft
  • 6593503 unreferenced object=/usr/lib/libz.so.1; unused dependency of /usr/lib/libfontconfig.so.1
  • 6594002 xvinfo update from 1.0.1 to 1.0.2
  • 6594444 xf86-video-savage-2.1.3
  • 6596598 Xorg headers should be delivered to build Xorg DDX modules without having to build the entire tree
  • 6579866 [tcde] cannot launch terminal in non-root administrative role workspace
  • 6571762 xrender.pc depends on non-existent x11.pc
  • 6594768 Package violation of file XKBsrv.h in SUNWxwinc and SUNWxorg-headers
  • 6593812 boing screensaver causes panic under Xen

Build 74

  • 6351300 libXcursor & xcursorgen [LSARC 2007/491]
  • 6598505 sizeof(I830DRIRec) not matched between i965_dri.so and intel_drv.so
  • 6598244 Xorg modularization: constype
  • 6598247 open source SUNWmisc subdir of X consolidation
  • 6601438 Xorg on Solaris should use freetype for TrueType font rendering
  • 6601918 Upgrade Xorg Intel video driver to xf86-video-intel-2.1.1
  • 6602983 xscreensaver won't build on snv_72 due to 6571762

Build 75

  • 6589352 Increase Mesa texture upper limit handling to 31 textures.
  • 6523003 Include fc-match in Solaris
  • 6470670 Include DejaVu fonts in Solaris
  • 6606099 Xft headers should be in /usr/X11/include, not /usr/X11/share/include
  • 6497669 tkdiff is crashing Xsun server
  • 6607872 Upgrade xf86-video-nv from 2.1.2 to 2.1.5
  • 6531358 xscreensaver fails to properly lock the screen during a typing break
  • 6580862 Unable to type in windows session with full screen mode when hotdesked into the same DTU on TJDS
  • 6595025 screen has 1 in lines that flash about the verical freq of the comsole vga mon
  • 6605582 nvidia driver doesn't recognize geforce 6150 chipset graphics hw
  • 6608421 nvidia driver looses mouse icon when Xserver or dtlogin restarts
  • 6606140 nvidia packages have incorrect dependencies in snv_73 and s10u4.

Build 76

  • 6601751 [X.Org Bug 12298] \*xfs\* Integer overflows in build_range()
  • 6601756 [X.Org Bug 12299] \*xfs\* swap_char2b() Heap Overflow Vulnerability
  • 6611790 glxgears, glxinfo, & ogl-select man pages should be in open-src tree
  • 6612322 libXcursor man pages should be delivered
  • 6586710 reference to PanoramiXGetInputFocus() in /usr/X11R6/lib/modules/extensions/libxtsol.so
  • 6604583 TsolExtensionInit: cannot load X Trusted Extensions error message showing up when time server starts
  • 6589829 X server should include zoneid of shared memory segment in access policy for MIT-SHM
  • 6614335 sparc X build fails due to missing glxinfo/glxgears man pages
  • 6425775 Xsun is blocked when a program rings keyboard- or audio-bell
  • 6614408 couldn't open face /usr/X11/lib/X11/fonts/TrueType/LucidaBrightDemiBoldItalic.ttf
  • 6611542 Xorg unable to load Type1 font module in snv_74
  • 6572087 ship Xvnc for Solaris [PSARC/2007/545]
  • 6615458 xfs(1) references the wrong FMRI for the X font server 

Build 77

  • 6545884 If xserver crashes repeatedly, keyboard is left in unencoded event reporting mode
  • 6523766 Xsun core dumps on X86 servers sometimes
  • 6617165 Upgrade from s10u2 and snv_60 to snv_75 had issues with SUNWxorg-mesa on x86
  • 6617243 Problems starting CD2 installer when using locale en_US.UTF-8 (only on x86)

Build 78  -- Source Drop 20071107

  • 6616521 typo in -httpd in vncserver man page
  • 6465564 Sun Ray hangs with 26D when running more than one "xset fp+ tcp/server1:7100"
  • 6624562 Xvnc should be built with Sun Studio
  • 6626260 32-bit & 64-bit versions of libXfont should be in the same package
  • 6616573 Ship RealVNC vncviewer in Solaris [LSARC/2007/625]

Build 79

  • 6627898 [tcde] xserver crashes on screensaver timeout being reached
  • 6570556 Xsun stops creating new client windows once its open file descriptors reach 256
  • 6627775 X11perf_shmem ShmPutImage and ShmPutImageXY std or xor aborts with setenv XSUNTRANSPORT shmem w :0.0
