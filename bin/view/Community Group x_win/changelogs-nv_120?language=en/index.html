X Change Logs: Nevada Builds 120-129

These changes went into the X Consolidation in Solaris Nevada Builds 120-129:

Build 120

  • 6858165 Upgrade Xorg to 1.6.2
  • 6825374 xscreensaver issues under Xorg
  • 6858998 Type1 prebuilt fonts should be moved to SUNWxsun-server too
  • 6859233 Removal of Xprint [PSARC 2009/294]
  • 6859906 Library links should be moved from SUNWxwrtl to packages with libraries
  • 6857559 Nevada build was broken because of 6849124. This is refix of 6849124.
  • 6860365 Remove lg3d library and headers
  • 6860461 exiting dtlogin causes the system to hang on ast on sparc
  • 6860468 move xc/lib/Xaw and xc/SUNWlib/3_1libXaw to open-src/lib/libXaw5 and open-src/lib/libXaw4
  • 6859883 Introduce XDM-AUTHORIZATION-1 support in libXdmcp
  • 6861501 X pkgs delivering to /usr/X11 should have SUNW_PKGTYPE of usr not ow
  • 6861571 fonts.aliases.all missing from F3bitmaps.
  • 6858498 Upgrade Mesa from 7.4.2 to 7.4.4
  • 6861947 deliver libdrm header files

Build 121

  • 6862443 Rip out the EOL'd speedo fonts
  • 6862466 xmh EOF  LSARC/2009/358
  • 6865425 Clean up some trash in the xc/fonts directory
  • 6741195 Remove Xsun cg6 driver [PSARC 2008/488]
  • 6862465 XIE EOL [LSARC/2009/359]
  • 6859039 XScreenSaver Symbolic Link Local Information Disclosure Vulnerability
  • 6862467 bdftosnf & showsnf EOF LSARC/2009/360
  • 6867643 X.Org mass minor app & lib module update: July 2009
  • 6866668 glxdri2.c should be built so DRI2 drivers can load
  • 6867733 Upgrade Xorg to 1.6.3 

Build 122

  • 6480003 Xorg should support virtual console on Solaris
  • 6824603 ACPI hotkey X support
  • 6827540 dispswitch: need to support display switch function key
  • 6852921 Recycling the Xorg server runs out of file descriptors in the keyboard driver
  • 6859397 xorgcfg & xorgconfig entries need to be removed from exec_attr
  • 6865266 Update libdrm to v2.4.12
  • 6867761 Xorg driver did not work well with new Intel build-in video card
  • 6868816 move xscreensaver from X consolidation to Desktop consolidation
  • 6872917 Xorg not loading nvidia driver when no /etc/X11/xorg.conf exists

Build 123

  • 6873773 Xorg hangs in ioctl(..., I_PUSH, "usbkbm") on VT switch
  • 6854377 Update deja-vu fonts to version 2.29
  • 6847399 gnome comes up in black & white with Trident Microsystems 3D Blade PCI/AGP graphics card in snv_116

Build 124

  • 6876992 [vconsole] Ctrl+Alt+F12 switches to blank console screen with hotkeys property turned-off.
  • 6854675 Update fontconfig to 2.7.1
  • 6876618 Upgrade Freetype2 to 2.3.9
  • 6877323 [CVE-2009-0946] security vulnerability in freetype 2.3.9 and earlier
  • 6874875 Xorg needs to have privilege to make PCITOOL_DEVICE_SET_REG and PCITOOL_DEVICE_GET_REG ioctl calls
  • 6875743 Xorg crashes after installing snv_122 on some x86 systems
  • 6882497 When you upgrade to X NV build 124, the nvidia driver is not used without xorg.conf
  • 6846735 Unplugging USB mouse during suspend kills X11
  • 6727075 Remove redxblue and SUNWlib/libpixmem from Closed source tree
  • 6863658 P3/S5 eVGA GTX 295 CO-OP not working as GLX nVIDIA card
  • 6879326 P2/S2 long running OpenGL programs can hang
  • 6879347 P3/S4 nvidia-settings crashes when directory name selected [OSOL 9548]
  • 6879355 P3/S3 Need support for MCP79 graphics
  • 6879359 P3/S4 Quadro FX 1700 display flicker/corruption in b117 [OSOL 9893]
  • 6879363 P3/S3 Need support for G105M in HP DV4 notebooks 

Build 125

  • 6727075 Remove redxblue and SUNWlib/libpixmem from Closed source tree
  • 6868776 fonts.conf customizations don't have any effect
  • 6869348 installing lang-support-all hijacks existing default fonts for some locales
  • 6881406 Synergy - Mouse/Keyboard sharing for Solaris, Linux, Windows, MacOS
  • 6885155 Xorg modularization: xdm 1.1.9 [LSARC/2009/512]
  • 6883157 xf86-video-nv 2.1.14

Build 126

  • 6872780 Memory fault(coredump) error in terminal when running x11perf trapezoid test case on SPARC machine
  • 6885229 Xorg server crashes in TryClientEvents()
  • 6858547 Xtsol policy needs to prevent exploit amplification
  • 6873789 Use XACE extension hooks to provide SUN_TSOL/TX functionalities
  • 6737079 Provide Xorg utility "xinput"
  • 6888175 SUNWsynergy: RUNPATH violations in /usr/bin/synergyc and synergys
  • 6865400 X consolidation does not build with Studio 12 Update 1 compilers
  • 6889905 Move rasterfile header & man page to open-src
  • 6890333 Upgrade xscope from 1.1 to 1.2 [LSARC/2009/549]
  • 6879897 SIGIO input handlers should be disabled by default until SIGIO handler issues are fixed
  • 6890443 Upgrade Xorg server from 1.6.3 to 1.6.5

Build 127

  • 6839874 Xserver crash when running x11perf shmputxy test case
  • 6889793 enable parallel builds of the X consolidation
  • 6892227 X builds should workaround OpenSolaris SGML catalog initialization bug
  • 6888696 A Java application using both awt/swing and eclipse/swt hangs in Xlibint.c.
  • 6886424 VT - keyboard and mouse do not work when switching to vt7
  • 6872597 need support for auburndale/havendale chipset
  • 6894403 Xorg should use /usr/X11/bin/xkbcomp instead of /usr/X11/lib/X11/xkb/xkbcomp
  • 6890536 [virtual keyboards] input does not work for java and xterm in Russian locale
  • 6892799 mouse driver closes fd while processing DEVICE_OFF
  • 6890351 Fontconfig version 2.7.3
  • 6895018 Remove the FT_CONFIG_ADD_FULL_HINTING patches

Build 128

  • 6896980 XReadScreen needs to be exported from libXext
  • 6898535 Upgrade VNC from RealVNC 4.1.3 to TigerVNC 1.0 [PSARC/2009/592]
  • 6899152 delete unused files from xc source dirs & /usr/openwin in packages
  • 6899419 Move libowconfig to open-src & /usr/X11
  • 6899422 Replace DPS client libraries with open source version from X11R6.9

Build 129

  • 6871838 `scanpci -v` produces "error ent" output on stderr
  • 6892784 Can't find a Primary video card  with dual graphics in Xorg server 1.6.1
  • 6899630 move libdga to open-src & /usr/X11
  • 6897086 Hangul and Hanja keys on Korean PC 105 keyboard are not recognized
  • 6902170 random hangs when running with GEM driver
  • 6902856 SUNWvncviewer and SUNWxnvc dynamic dependencies not captured
  • 6902871 SUNWxwdxm dynamically linked files not found
  • 6902873 SUNWxwplt dynamically linked file not found
  • 6737076 Bring in Xorg's xcompmgr window manager
  • 6884183 GNOME is really slow in OpenSolaris build 122+ running in VirtualBox
  • 6896574 Font repackaging and upgrade to X11R7.5 fonts
  • 6904057 Fontconfig configuration in snv_127 is broken
  • 6898754 Performance regression found on snv126 + Xorg B127 on Intel G33
  • 6869348 installing lang-support-all hijacks existing default fonts for some locales
  • 6900046 wqy-zenhei.ttf is used for Japanese locale in snv_127
  • 6866513 shaped window resizing causes almost entire screen to flicker rapidly
  • 6883083 OpenGL 3.2 support in NVIDIA graphics driver [LSARC/2009/569]
  • 6890469 NVDAgraphics package dependency on dynamically linked lib not captured
  • 6892748 Add ABI 6 / X.Org 1.7 support to bundled NVIDIA graphics driver
  • 6893016 Tecra M10 non-native resolutions produce blank displays
