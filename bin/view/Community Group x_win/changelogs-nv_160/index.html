Nevada Builds 160-169

X Change Logs: Nevada Builds 160-169

These changes went into the X Consolidation in Solaris Nevada Builds 160-169:

Build 160

  • 7010624 X11R7.6 Update: Application, Utility, & Data modules [PSARC/2011/033]
  • 7013236 Update pixman to 0.20.2
  • 7019195 stop embedding hg id/tag & build date in mcs information in X
  • 7010623 X11R7.6 Update: Library modules
  • 7017145 Xorg sometimes fails to reply to a client that grabbed it

Build 161

  • 7019730 xcb build fails with Studio 12.2 compilers
  • 6988508 Restore noexbss & bssalign mapfiles to X builds after fix for 6988300
  • 7020776 X gate build fails in proxymngr when lbxproxy is not installed
  • 7001077 Can not startup xinit on sparc box, with non-root user
  • 7013236 Update pixman to 0.20.2 (update license notice)
  • 7022271 set-elf-comments.pl can induce race conditions in X gate builds
  • 7022321 suntouch-manpages.pl should stop trying to touch the shadows
  • 7022455 libfontenc.so.1 is still using /usr/lib/X11/fonts/encodings

Build 162

  • 6983602 Xorg 1.9 integration [PSARC/2011/008]
  • 7012628 Intel Graphics 2009Q4 updates (xf86-video-intel 2.10 & mesa 7.8.2)
  • 7014789 CDE font aliases missing on Solaris 11
  • 7025232 libxcb pkg should have a devel dependency on libpthread-stubs pkg

Build 163

  • 6966962 libpciaccess should support multiple PCI segments
  • 6991718 Removal of 32-bit X servers [PSARC/2010/382]
  • 7023672 The vncserver is unable to start a typical gnome Solaris 11 desktop, defaults to xterm
  • 7026635 Sun-Gothic and Sun-Michou fonts should be added in fonts.conf
  • 7030089 X fd_sets use too many fd's in 64-bit mode
  • 7031334 Programs linked against libSM on S8 fail to run on snv_160
  • 7031753 Updated xorg-video-savage obsoletion package needs build number bump

Build 164

  • 7026635 Sun-Gothic and Sun-Michou fonts should be added in fonts.conf
  • 7032104 Unable to start vncserver - Assertion failed
  • 7021972 rstart should be removed from Solaris [PSARC/2011/072]
  • 7010836 Package manifests in X gate should use .p5m naming convention
  • 7012056 Xlib no longer switches to local sockets when DISPLAY specifies local hostname
  • 7012058 libxcb doesn't support multilevel unix domain sockets
  • 7022433 CVE-2011-0465 Root execution vulnerability via xrdb by DHCP and XDMCP
  • 7027761 Matrox G200eR2 (102b.534) not working with Xorg
  • 7032292 starting compiz causes a xserver crash with 162
  • 7035299 clean up CONFIG_SHELL in X makefiles
  • 7035408 Upgrade libX11 to 1.4.3 for deadlock & static analysis fixes

Build 165

  • 7037681 firefox-xrx plugin should be 32-bit
  • 7038737 sparcv9 Xlib looking in wrong path for i18n loadable modules
  • 7039092 Move efb module for Xorg to X consolidation
  • 7039328 Move SPARC Graphics utilities to X consolidation

Build 166

  • 7017116 xf86-video-ati 6.14.1
  • 7038208 EOF xrx ("Broadway") [PSARC/2011/203]
  • 7041658 X packages should not deliver dir actions for mdb directories
  • 7042476 Xorg 1.10 & associated module updates [PSARC/2011/214]
  • 7042918 Update xcb-util to version 0.3.8 [PSARC/2011/215]

Build 167

  • 7041591 duplicate and missing dependencies in x11 IPS packages
  • 7043714 X pkgs need to be pkglinted
  • 7047071 xcolor segfaults on startup
  • 7047456 X could do better at building & packaging python bits

Build 168

  • 6960231 xdm login fails with: pam_unix_cred: cannot load ttyname with c2 audit loaded
  • 7023890 resize(1) needs to move out of pkg:/terminal/xterm
  • 7047054 Removal of hwc [PSARC/2011/230]
  • 7048514 X gate should use pkgrepo command
  • 7048963 Build race in open-src/driver/efb
  • 7049296 exec_attr entry should be provided for fbconfig
  • 7049366 fbconfig -list can print invalid characters when failing to open a device
  • 7049367 fbconfig not working after conversion to IPS packages
  • 7050250 xsetroot crashes Xorg server
  • 7051435 xorg-server 1.10.2

Build 169

  • 7005670 Unable to type L and R with acute on Slovak keyboard
  • 7008033 dead keys do not work in portuguese brazillian keyboard layout
  • 7010804 Xorg should audit connections and disconnections.
  • 7026593 Add "RO" to res1 field of prof_attr.d & exec_attr.d files in X
  • 7054723 X consolidation should move to perl 5.12
