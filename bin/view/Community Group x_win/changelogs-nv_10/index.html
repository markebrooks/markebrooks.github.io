Nevada Builds 10-19

X Change Logs: Nevada Builds 10-19

These changes went into the X Consolidation in Solaris Nevada Builds 10-19:

Build 10

  • 6226733 ISO_Lock keys missing from IsModifierKey macro in Xutils.h
  • 6224159 Xsun on x86 diddles with console device when started on non-console.
  • 6206898 [Xorg] B&H Hebrew (ISO8859-8) Type 1 fonts are missing Hebrew glyphs
  • 6223269 scanpci should link with shared libscanpci, not static
  • 6227475 Some Asian & European keyboards don't work in Xorg
  • 6227952 Update from Xorg 6.8.2 rc3 to final release

Build 11

  • 6229445 fc-cache crash
  • 4784055 locked-screen dialog is inaccessible to Gnopernicus

Build 12

  • 4751462 [Solaris Xsun/Xinerama] Cannot move the cursor across screens with mousekeys
  • 5074303 lbxproxy core dumps on exit if clients have had authentication failures
  • 6241936 [CAN-2005-0605] XPM buffer overflow in libXpm
  • 6231515 Bug fix of Additional overflows in XPM code causes regression
  • 6237957 [Xorg bug \#2533] Feb. 2005 nv chipset update
  • 6244025 StringDefs.h & Shell.h in Xorg tree incompatible with Solaris libXt

Build 13

  • 6248445 3rd-button emulation problem with Xsun
  • 6247062 Resource file search path needs to include /usr/X11/lib/X11
  • 6248453 Integration of bug fix for _XkbReadVirtualModMap() in libX11
  • 6248417 Update libXmu to X11R6.8 and add /usr/X11/include/X11 in bitmap file search
  • 6219170 Xsun needs to support absolute mouse scaling
  • 6225953 Error in fsadmin(1M) manual page.
  • 6247100 Xorg man pages missing External stability notice
  • 6219169 Xorg needs to support absolute mouse scaling
  • 6207005 Xorg should implement agpgart wrappers instead of simply returning FALSE
  • 6248315 X client support for XFree86 extensions
  • 6251506 NV ddx should set fptweak when it can

Build 14

  • 6251577 Xorg: wchar_t redefined in Xlib.h, causing type conflict with
  • 6251042 xserver/devices_x86 - Update i810 Xorg driver to Mar-2005 version
  • 6212502 BGR-RGB swap in RENDER
  • 6237901 Fix for pam code

Build 15

  • 6237901 Ldap and Gnome xscreensaver authentication failure
  • 6261066 amd64 libfreetype.so can't load due to text relocation errors
  • 5092820 JDS 3 BETA 1:  xscreensaver not compiled properly: missing feature
  • 6266780 symbol versions incorrect for 6248417 additions to libXmu
  • 6265045 buffer overflow in Xsun
  • 6265634 "ddxSUNWmouse: sunMouseEnqueueEvent: unrecognized id 32646 on /dev/mouse" visible on a few installs
  • 6251042 Update i810 Xorg driver to Mar-2005 version
  • 6193797 Xorg server needs a better configuration mechanism
  • 6241320 "Emulate3Buttons" option in Xorg does not work correctly with a USB mouse
  • 6262645 xorgcfg: Chipsets missing for some drivers in Card screen
  • 6264498 [Xorg bug \#3095] Toshiba Tecra M3 display "blooms" instead of blanks when DPMS turns off the screen

Build 16

  • 6228023 xscreensaver segfaults when XInputExtension is missing
  • 5077974 GNOME Bug 147583: Screen Lock unlocks without authentication
  • 5083155 [Cinn Solaris] Unable to unlock screen when running dual-head magnification
  • 5092820 JDS 3 BETA 1:  xscreensaver not compiled properly: missing feature
  • 6182506 Screensaver dialog isn't magnified properly
  • 6216985 Xsun & Xorg Xinerama protocols are incompatible with each other
  • 6226717 memory leak in _XlcInitCTInfo with japanese locale
  • 6240938 xscreensaver-lock's password timer needs to to be reset for each key
  • 6266805 X libraries missing version symbols on amd64
  • 6267306 new image should be included in xscreensaver dialog
  • 6269444 snv_14 xscreensaver prints out gratuitous debug messages after authentication
  • 6264856 Turn on Mesa Distribution for Solaris, ARC 2005/109
  • 6252544 kana_I cannot be input on Xorg with Japanese type6 keyboard.

Build 17

  • 6278634 Instead of initial GUI login screen, screen flashes alterately black then white
  • 6275666 Xorg loops in poll when VUIDSFORMAT fails on one mouse in a multi-mouse system
  • 6277477 xorgconfig should enable glx by default
  • 6261221 view page source causes mozilla crash on S10 b74L2a
  • 6276256 PCF fonts should not be compressed in core X11 font packages

Build 18

  • 6226717 memory leak in _XlcInitCTInfo with japanese locale
  • 6269938 Solaris 8 and 9 Xtest call is not Xinerama aware
  • 6278131 twm dumps core with high-numbered mouse button events
  • 6279680 ddxSUNWmouse: couldn't set absolute mouse scaling resolution on /dev/mouse
  • 6280222 solaris 10 X86 xvfb Virtual Frame Buffer in a local zone

Build 19

  • 4697054 Xsun & Xvfb man pages need update for PSARC 2002/282
  • 6276301 Font software and fonts should be in separate packages
  • 6278337 i810 driver update for i945G & E7221 support and other enhancements
  • 6278365 glxinfo & glxgears missing from packages
  • 6281312 xorgcfg saving configuration mechanism behaves differently on different hardware
  • 6281313 xorgcfg has problems in configuring new layout
  • 6290756 Update ati Xorg drivers to Jun-2005 version
  • 6292556 Upgrade FreeType to 2.1.10
