X Change Logs: Nevada Builds 60-69

These changes went into the X Consolidation in Solaris Nevada Builds 60-69:

  • 6524801 Xorg on SPARC should be setgid root not setgid bin
  • 6525684 Xtsol module build changes needed for Xorg 7.2 integration
  • 6525733 Xorg 7.2 crashes on startup when Xtsol module is present
  • 6525711 64-bit Xvfb missing from SUNWxorg-server packages on x86
  • 6524795 xf86-video-vmware 10.15.0
  • 6525814 x11-server list of allowed servers needs to be updated for Xorg 7.2 integration
  • 6525243 Nevada B58 32-bit Xorg crashes on a VMware VM
