Nevada Builds 110-119

X Change Logs: Nevada Builds 110-119

These changes went into the X Consolidation in Solaris Nevada Builds 110-119:

Build 110

  • 6526791 xscreensaver and X.org need to change timeouts for MOU4
  • 6770058 FcFontSetSort will dump if first parameter is NULL
  • 6737254 kk_KZ.UTF-8 locale: In Java applications changing keyboard layout with gimlet does not work
  • 6808868 Mesa should be compiled with -xspace to reduce binary sizes
  • 6809017 X builds should use SUNWlynx now that it's integrated, instead of SFWlynx
  • 6809582 dispswitch should be moved to its own source directory
  • 6810801 Xorg doesn't start on 8xx chipsets with intel drv 2.4.2
  • 6809337 SUNWxwcft/SUNWxwoft own duplicate files
  • 6810319 bring nevada font packages closer to opensolaris ones
  • 6809578 Xserver script adaptation to a world without Xsun (6805131) needs to adapt to miniroot case
  • 6803790 Add Google Droid fonts
  • 6810911 Update deja-vu fonts to version 2.28 
  • 6809538 Update Cg Toolkit to 2.1 - Feb 2009 release

Build 111

  • 6812503 Missing fonts after update to 109
  • 6809309 Add new utf8 locales supported by Xlib
  • 6805297 Xorg consuming too much CPU when GNOME calls SYNC extension
  • 6813198 [defect 7060] glxinfo -i is coring
  • 6813925 Xorg needs to catch ld.so.1 failure so it can close down devices cleanly
  • 6809996 xrx needs to include npfunctions.h instead of npupp.h since Mozilla renamed it
  • 6736157 [A11Y]Security problem when desktop a11y support is turned on.
  • 6814890 Motif links in /usr/\{lib,include\} should be moved from SUNWxwrtl/SUNWxwhl to SUNWmfrun [X side]
  • 6815064 Merge SUNWxwhl into SUNWxwinc and change link target to /usr/X11/include
  • 6757448 xscreensaver doesn't notice XRandR resize events
  • 6815181 RRSelectInput should check for DixRecieveAccess not DixWriteAccess
  • 6815700 X consolidation should be built with -xspace to reduce binary size
  • 6802980 [tjds] labeled workspaces not starting
  • 6638904 Instructions for enabling remote access to X server are incomplete
  • 6816129 Xorg man page improvements for 2009.06 release
  • 6810237 SUNWxwfnt upgrade is broken in 109 due to issues with bad \*ph files delivery
  • 6816942 Add EOF notice to xorgcfg & xorgconfig man pages
  • 6819154 cannot do default install osol_108 or later with Intel GM45
  • 6820467 xscreensaver unlock logo update for OpenSolaris 2009.06

Build 111a

Build 111a is a respin of build 111 to fix several bugs for the OpenSolaris 2009.06 release.   Since it occurred after Build 112, some of these fixes are not included in Build 112, but will be in Build 113.

  • 6819661 [nevada][tjds] many desktop applications do not launch in TX
  • 6824137 No keyboard focus on desktop from live CD with b109
  • 6822628 constype returns unexpected value for nvidia card, ogl-select picks generic GLX extension in snv_111
  • 6820699 Update bundled NVIDIA graphics driver to 180.44  including://
  • 6794113 add support for nvidia graphics card of id: pciex10de,658
  • 6683369 nv_devmap errors under xVM dom0 on i865 motherboard
  • 6806078 compiz dumps core under the tender mercies of libumem
  • 6812940 6747167 breaks menu on Solaris 10 updates
  • 6824533 nv_quiesce() implementation does not adhere to quiesce(9E) spec
  • 6825640 GeForce 9600 GSO device alias missing

Build 111b - OpenSolaris 2009.06 Release

Build 111b is a respin of build 111a to fix a few last bugs for the OpenSolaris 2009.06 release.   Since it occurred after Build 114, some of these fixes are not included in the main build stream until Build 115.

  • 6835232 remote Motif app crashes on keyboard input in snv_111a
