Communication

If you meet any problems or have any suggestions for Mozilla applications on OpenSolaris, the best way would be through the mailing list, the address is desktop dash discuss at opensolaris dot org.

Get Involved

 - For any Mozilla bugs related to OpenSolaris, please direct to desktop dash discuss at opensolaris dot org, or report the bug in Mozilla's bugzilla.

 - Please help us keep eyes on the Firefox Tinderbox trees and Thunderbird Tinderbox trees. Any fire for the trees, please report to desktop dash discuss at opensolaris dot org, or even better, to fix the broken issue in the Mozilla community.

 - If you want to join the Mozilla development on OpenSolaris or just want to help out, bugs triage is a fairly easy task that always needs to be done. The list of the OpenSolaris UNCONFIRMED bugs is here.

 - The next step is fixing the properly triaged bug reports. All the patches should go to upstream Mozilla community for review first. OpenSolaris will sync with the stable release from Mozilla community.

 - The contribution for product Quality Assurance is always welcomed by using the Mozilla Litmus test platform. The cutting edge users can go with the alpha/beta releases or even the nightly builds. Please direct all the questions to desktop dash discuss at opensolaris dot org, and report the bug in Mozilla's bugzilla.

 - If you're interested to take the ownership to provide contributed builds for SeaMonkey / XULRunner / Sunbird / Spicebird / BlueGriffon / Instantbird, or Mozilla add-ons / Mozilla labs project, please direct all queries to desktop dash discuss at opensolaris dot org. Four OpenSolaris/Solaris 10 Virtual Mahines in China (x86/SPARC) will be provided to the desktop community. You can apply for the access with a proposal on how the machines will be used (for which project, for how long, what's the plan etc.) to desktop dash discuss at opensolaris dot org. The access will be evaluated/renewed every 3 months. Please note that the usage policy for the machines is still under discussion and not finalized yet.
