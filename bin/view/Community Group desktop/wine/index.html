 Wine is a compatibility layer for running Windows applications on top of Unix and Unix-like operating systems. Wine re-implements the Windows API and does not use any Microsoft code, however, where necessary it is possible to use native Windows DLLs from your licensed copy of Windows. Wine provides both a development toolkit for porting Windows source code to Unix as well as a program loader, allowing many unmodified Windows programs to run on x86-based Unixes. Wine is licensed under the GNU Lesser General Public License version 2.1.

Get involved!

 There is a team of people who are passionate about running Windows Applications on OpenSolaris, we discuss things in the Desktop Community Discussions List if you want to join us. Thanks to the network effect you are even helping us out just by running Wine on OpenSolaris. If you'd like to do even more look at the list below for some suggestions on how you can make a difference and share with other people.

Non Technical

  • Use Wine on OpenSolaris
  • Blog about your experiences
  • Submit RFE's for Wine and OpenSolaris
  • Advocate/evangelise
