The OpenSolaris Desktop Community mailing list

 Solaris, and OpenSolaris, have traditionally been recognized as a server based operating system. While this still remains true to today, the importance of the desktop has grown over the last couple of years. The Desktop Community brings desktop users and developers together from right around the world, 24 hours a day, 365 days a year, to collaborate on support for the OpenSolaris Desktop.

 The Desktop Community will act as an umbrella for many desktop related projects with a focus on OpenSolaris - everything from GNOME, KDE, Mozilla, and many, many others. Join our mailing list and contribute today!

Community Goals

  • Build an active community of users running desktop based applications on OpenSolaris
  • Build an active community of developers creating new, and porting existing applications to OpenSolaris
  • Challenge the belief that OpenSolaris is just a server based operating system
