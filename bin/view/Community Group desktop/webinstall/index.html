Package Manager Web Install

Web Install

Simple one click addition of repositories and installation of packages from the web on OpenSolaris 2009.06.
 Clicking on a p5i file on your OpenSoalris 2009.06 desktop or on a web server with the p5i mime type registered, launches Package Manager in WebInstall mode. It then parses the p5i file to see which repositories to add to your system if they are not already there and to install any specified packages for these repositories.
 At this point you can decide to proceed or cancel.

 To try it out:

  • Click on test.p5i. This will add opensoalris.org, sunfreeware and contrib if they are not already on your system and install a few packages from them.
  • Click on extra_flash.p5i to install Flash plugin from extra repository.
  • Click on cbe.p5i to install packages needed to build Gnome on OpenSolaris. No Publisher is specified in this p5i file so packages are taken from the default Repository which for most users will be an OpenSolaris core repository

 We plan to move the following p5i files to OpenSolaris Package Repositories when 2009.06 is released to allow easy addition of repositories for different users.

  • Click on user.p5i to add repositories that are of use to users.
  • Click on developer.p5i to add repositories that are of use to developers.

Mime Type

To have a one click web install, web servers hosting the p5i files need to have the new p5i mime type, application/vnd.p5i.info registered.

 If you are running apache then an AddType entry needs to be added to its httpd.conf (/etc/apache2/2.2/httpd.conf) mime_module section:

 <IfModule mime_module>
 AddType application/vnd.pkg5.info .p5i
 </IfModule><br/>
