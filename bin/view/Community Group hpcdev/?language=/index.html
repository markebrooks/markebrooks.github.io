Whats New ...

 This is a community for anyone interested in High Performance Computing (HPC) on OpenSolaris. Its purpose is to:

  • Provide a home for the OpenSolaris HPC Developer distribution project,
  • HPC Deployer project,
  • Highlight existing features and future enhancements relevant for high-performance computing,
  • Provide a forum to discuss issues and possible solutions to problems faced by HPC developers, and
  • Provide pointers to pertinent resources and tools currently available

DTrace for Open MPI

 DTrace is a comprehensive dynamic tracing framework for OpenSolaris. It provides a powerful infrastructure to permit administrators, developers, and service personnel to concisely answer arbitrary questions about the behavior of the operating system and user programs.

 This project proposes to add DTrace probes into the MPI library, leveraging the PERUSE / OTF instrumentation (Vampirtrace) to ultimately allow tracing of key point-to-point communication events at the ClusterTools Point to Point Messaging Layer (PML)

 Traceable events will include:

  • Initialization, Completion and Notify of Requests
  • First and successive actions of sending data on a network
  • Arrival of messages
  • Inserting, matching, and removal of Receive requests on posted queue
  • Inserting, matching and removal of Receive messages on unexpected queue
  • Successive data fragments received for incoming messages
