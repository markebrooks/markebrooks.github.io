Zone Manager

Overview

The purpose of this project is to simplify Solaris zones creation and management down to a single non-interactive command line tool.  The Zone Manager (zonemgr) should enable administrators to do 90% of zone creation and management.  Note however, that this project is in NO WAY intended to replace the existing zonecfg/zoneadm commands but rather to supplement them.

Project Goals

  • Simplify zones creation
  • Simplify zones management
  • Incorporate Solaris features into zone creation/management
      such as resource controls, minimization, service lockdown,...
  • Incorporate external features such as network package repository based package installation.

This project will continue so long as there is sufficient interest to keep it around.

Status

  • Version 2.0.7.1 is in production.
  • Version 2.0.6 is depricated.
  • Version 1.8.1 is depricated and has been removed.

Documentation

You can run zonemgr with the -h option to view the help information.

Alternatively, you can see the version specific on the
documentation page.

You can also use the following presentations to get a basic overview of the Zone Manager.  Use it to educate your friends and family on the ways of the Zone Manager.

  • Zones Automation In The Real World ODP PDF
  • Zone Manager Introduction ODP PDF

Support

Although the intent of this project is to simplify zones management,
it is not a supported part of Solaris.  Thus the Zone Manager and its associated scripts are NOT in any way supported.  However, best effort is made by the contributors to address bugs as soon as possible.

Contributions and Reporting Bugs

Discuss issues or feature requests on the Zones-Discuss community.  You can then submit bugs here and requests for enhancement here.
