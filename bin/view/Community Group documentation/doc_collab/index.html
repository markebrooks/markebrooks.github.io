Documentation Contributor Resources

 We encourage OpenSolaris project participants to join the documentation community and contribute to the documentation effort. There are many ways you can participate:

  • Review documentation for clarity, usability, and technical accuracy.
  • Provide artwork or illustrations.
  • Help define real-world use cases that need better or more focused documentation.
  • Share how you use OpenSolaris by writing "how-to" procedures in the documentation wiki collaboration/contribution space.
  • Become a moderator in the documentation wiki.
  • Work in collaboration with Sun sponsors on feature documents in the OpenSolaris Documentation Collection and the OpenSolaris OS man pages.
  • Support the tools used to work on feature documents outside Sun's firewall.

 To learn more, refer to the following pages.
