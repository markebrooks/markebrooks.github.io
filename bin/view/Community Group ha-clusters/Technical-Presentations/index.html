Technical Presentations

Technical Presentations

 (Note that the presentations are in a variety of languages, including English, German, and Polish).

  1. Agent development for SUNW.gds based agents:
Title:How to wrap applications into a non-global zone
Author:Thorsten Früauf, Neil Garthwaite
Abstract:This presentation was given to the Cluster Technology Forum in August 2005. At that time, SC 3.1 08/05 with the HA-Container agent was just released. The presentation explains how some SUNW.gds-based agents are changed in order to work in the traditional RGM model as well as within a non-global zone, which is under HA-Container agent control. The content still applies to the GDS-based agents today with Sun Cluster 3.2 and Open HA Cluster. It helps to understand the code flow for the different deployment scenarios with zones.
  1. Explanation of the two Solaris Zones models within Sun Cluster:
Title:Sun Cluster and Solaris 10 Containers
Author:Thorsten Früauf, Detlef Ulherr
Abstract:This presentation was given to the Customer Engineering Conference (CEC) in October 2006. It explains the two principle models on how Sun Cluster 3.2 interacts with Solaris Zones. Both models are explained and contrasted with decision criteria as to which model to choose, based on application and service requirements.
  1. Workshop on how to use the GDS coding template:
Title:Sun Cluster Agent Development - Advanced GDS Methodology
Author:Detlef Ulherr, slightly revised by Thorsten Früauf
Abstract:This presentation was given to the Cluster Technology Forum in March 2007. It was also delivered several times as a workshop for internal and external engineers, as it introduces agent development for Solaris Cluster and describes in detail how the generic data service and the coding template work. The presentation also contains an example for self practice. After working through this workshop you will be able to write your own custom agent. The coding template includes experience and feedback from several engineers and got enhanced during its use as base for most GDS based agents delivered within Open HA Cluster.
  1. Tutorial Part 1 about Open High Availability Cluster (OHAC) from the German Unix User Group (GUUG)"Frühjahrsfachgespräche 2008" at the university of Munich [German]:
Title:GUUG-Frühjahrsfachgespräch 2008 Tutorium Teil 1: Open High Availability Cluster [German]
Author:Thorsten Früauf
Abstract:This part of the tutorial was given at the German Unix User Group (GUUG)"Frühjahrsfachgespräche 2008" event in March 2008. It gives an introduction into the Solaris Cluster architecture, an overview as of what OpenSolaris and the HA Clusters community group is, an overview about the active OHAC projects and instructions on how to get started with building the agent and Geo Edition source and how to participate further. Finally it has a short introduction into agent development by using the GDS coding template. The presentation slides and speaker notes are in German.
  1. Tutorial Part 2 about Flying Container from the German Unix User Group (GUUG)"Frühjahrsfachgespräche 2008" at the university of Munich [German]:
Title:GUUG-Frühjahrsfachgespräch 2008 Tutorium Teil 2: Flying Container [German]
Author:Hartmut Streppel
Abstract:This part of the tutorial was given at the German Unix User Group (GUUG)"Frühjahrsfachgespräche 2008" event in March 2008. It gives an introduction into the concept of Flying Container, explains how to setup the HA Container Agent and how to expand the configuration with a setup for the Geo Edition. Part of the presentation is a detailed walk through the necessary steps and show them live on a Geo Cluster running within the Vision Center at Sun in Munich. The presentation slides are mainly in German, parts are in English.
  1. Open HA Cluster and project Colorado [Polish]
Title:Open HA Cluster (Polish)
Author:Piotr Jasiukajtis
Abstract:Introduction to Sun Cluster, OHAC project and the future of Open HA Cluster with project Colorado.
