Open High Availability Cluster (OHAC) is the open-source code base of Solaris Cluster1, a high availability (HA) clustering solution from Sun Microsystems, Inc. Solaris Cluster includes the core clustering framework, a suite of HA agents for various applications, an automated test framework, and a disaster recovery extension.

Different subsets of the code are being released in stages through Open HA Cluster. The open-source initiative shall develop in full view of the community, and could form the basis of future versions of the Solaris Cluster product group.

How Is Open HA Cluster Different From Solaris Cluster?

 Solaris Cluster is solely an end-user product, whereas Open HA Cluster has an open-source code base, along with build tools necessary to develop, build, and use the code. It provides infrastructure for communicating and sharing related information, and provides a platform for collaboration by the community to add new cutting-edge features to the code base.

 Sun support is available for the Open HA Cluster 2009.06 binary release. This release contains a subset of the functionality in the Solaris Cluster product as well as some new features that are introduced in this release. See the Open HA Cluster 2009.06 Release Notes for more information.

 Support for any code that is not part of the Open HA Cluster 2009.06 release as well as the source form will continue to be provided by the community.

 The Solaris Cluster product group is branded, tested, maintained, and supported as a Sun product. Future releases of Solaris Cluster could be built from the Open HA Cluster source code, but will be completely supported in the same manner as current versions of the Solaris Cluster products. However, the intent is to release as much of the existing source code as possible through Open HA Cluster, and to have future development be directed through the HA Clusters community. At any time, there could be differences in the content of Open HA Cluster and the Solaris Cluster products.

 At this time, upgrading or downgrading between Open HA Cluster source code and the Solaris Cluster product is not supported.

Getting Involved

Interested? You will find here all the information and support to get you started. You can start by using the product and experience its amazing features, or get adventurous and start hacking. On this website, you will find help on how to build, test, and deploy Open HA Cluster, information to help you dive into the code base, and instructions on how to make contributions to the code base.

  • Start by browsing the Open HA Cluster source code.
  • Download and build the source code.
  • If you want to dig deeper, go to the Open HA Cluster documentation page to access detailed documentation, slides, TOIs, and more.
  • Meet and talk with people in the ha-clusters:discuss discussion group as well as various other OpenSolaris discussion groups.
  • If you notice any problems, please report them.
  • If you want to contribute, follow the contributor instructions or jump straight to a list of potential projects.
  • If you are stuck or feel lost, read the FAQ. If the answer's not there, search the ha-clusters:discuss archives or send mail to ha-clusters-discuss@opensolaris.org.

1The Solaris Cluster product group contains Sun Cluster software, Sun Cluster agents (data services), and Sun Cluster Geographic Edition software.
