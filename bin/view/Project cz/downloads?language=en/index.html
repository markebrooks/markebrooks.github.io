Používáme OpenSolaris

Chcete-li vyzkoušet distribuci založenou na OpenSolarisu, pak si vyberte jednu z těchto variant:
Solaris Express Community Edition (CE) – DVD: stáhnout | CD: stáhnout

Solaris Express CE je binární distribuce od firmy Sun pro vývojáře (kódové jméno "Nevada"). Obsahuje nejnovější, Sunem nepodporovanou distribuci OpenSolarisu, ale i další technologie, které nebyly dosud zahrnuty do zdrojových kódu OpenSolarisu. Vývojáři mohou sestavit OpenSolaris ze zdrojových kódů, když použijí tuto distribuci jako základní systém. Je aktualizována každý druhý pátek.

Solaris Express Developers Edition (DE) - stáhnout | více informací

Solaris Express DE je oficiální distribuce OpenSolarisu od firmy Sun, ale i další technologie, které nebyly dosud nezahrnuty do zdrojových kódu OpenSolarisu. Sun nabízí omezenou podporu pro tuto distribuci. Je primárně určena pro uživatele Solarisu, aby si mohli vyzkoušet nejnovější Sunem nabízené technologie, které se mohou stát součástí Solarisu. Distribuce Solaris Express DE je aktualizována každé tři až čtyři měsíce a je volně dostupná ke stažení.

BeleniX – stáhnout | více informací

BeleniX je UN*X distribuce, která je postavena na zdrojovém kódu OpenSolarisu. Momentálně se jedná o LiveCD (i LiveUSB) distribuci. V budoucnosti by z ní však mohla vzniknout kompletní distribuce, kterou bude možné nainstalovat na pevný disk.

marTux – stáhnout | více informací

marTux je první ne Solaris Express distribucí pro procesory SPARC (sun4u nyní, sun4v později).

NexentaOS – stáhnout | více informací

NexentaOS je kompletní distribuce založena na GNU open-source, který se spoléhá na jádro a prostředí OpenSolarisu. NexentaOS spojuje OpenSolaris (SunOS kernel) a Open Source Software (OSS) aplikace a kombinuje tak to nejlepší z obou světů.

SchilliX – stáhnout | více informací

SchilliX je OpenSolaris LiveCD distribuce pro x86, x64 a EM64T architektury. Současná verze je založena na Nevada verzi 17 a vyšších.

Učíme se OpenSolaris

OpenSolaris projekt vytvořil Startovací balíček pro vývojáře, kteří se chtějí naučit, jak nainstalovat, nastartovat a spustit OpenSolaris. Startovací balíček zahrnuje porovnání OpenSolarisu s jinými operačními systémy.

Sestavíme si OpenSolaris

OpenSolaris se skládá z mnoha různých softwarových komponent a nejvýznamnější komponenty jsou označeny jako "konsolidace". Konsolidace obsahující základní operační systém a podporu sítě (Operating System/Networking (ON)) je základem OpenSolaris systému. K sestavení OpenSolarisu ze zdrojového kódu, potřebujete nejprve nainstalovat vhodnou distribuci OpenSolarisu, která je nyní limitována na Solaris Express CE.

Konsolidace OpenSolarisu

Projekt OpenSolaris se skládá z mnoha softwarových komponent. Nejvýznamnější komponenty jsou označeny jako "konsolidace". Konsolidace obsahující základní operační systém a podporu sítě (Operating System/Networking (ON)) je základem OpenSolaris systému.

Operační systém a podporu sítě (ON)

ON zahrnuje jádro, síťové služby a systémové knihovny a příkazy.

Další OpenSolaris konsolidace:

Aplikační server

Aplikační Server ze softwarové skupiny Sun Java System.

Produkční vývojové nástroje (Developer Product Tools (DevPro))

Tato komponenta má více částí: systémová matematická knihovna, media knihovna, microtasking knihovna, SCCS, make a C++ běhová knihovna. Některé jsou dostupné ve zdrojovém kódu, jiné jen binárně.

Dokumentace (Docs)

Vývojová a administrátorská technická dokumentace. Některé knihy jsou dostupné i zdrojové formě. Pro více informací se podívejte se na stránku projektu.

Národní podpora (Globalization (G11N))

Mezinárodní a lokalizační podpora.

Instalace (Install)

Instalace OpenSolarisu a nástroje pro správu balíčků.

Grafický desktop (Java Desktop System (JDS)/GNOME)

Bezpečné a komplexní řešení pro desktop.

Java platforma (Java SE)

Binární balíčky pro Java Development Kit (JDK) a Java Runtime Environment (JRE).

Manuálové stránky

Zdrojový kód referenčních manuálových stránek Solarisu.

Server fronty zpráv (Message Queue/JMS)

Server fronty zpráv ze softwarové skupiny Sun Java System. Je dostupný v open-source podobě.

Síťová datová úložiště (Network Storage (NWS))

Podpora pro síťové datové úložiště.

Solaris FreeWare (SFW)

Open-source software, který je dodáván spolu s Solaris/OpenSolaris.

Podpora grafiky na SPARC platformě

Ovladače pro grafickou podporu na SPARC platformě jsou dostupné v binární formě.

Testování

Testovací přípravky a nástroje pro OpenSolaris.

X Window System (X11)

X11 software.

Další OpenSolaris konsolidace budou dostupné i v budoucnosti. Pro více informací se podívejte na plány do budoucna.

Nástroje

Pro kompilaci OpenSolaris ze zdrojového kódu doporučujeme používat kompilátory Sun Studio 11. Některé konsolidace pracují i na podpoře kompilátorů GCC jako doplněk k Sun Studiu. Více na stránkách jednotlivých konsolidací.
