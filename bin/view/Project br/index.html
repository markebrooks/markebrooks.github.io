Bem-vindo ao Portal Brasileiro de OpenSolaris

O que é o OpenSolaris ?

OpenSolaris é um sistema operacional, um projeto de código aberto, um portal e uma comunidade internacional de desenvolvedores e usuários. Os objetivos do projeto são inovação, colaboração e a extensão da tecnologia OpenSolaris.

  • Código fonte OpenSolaris: O código fonte é a base para desenvolvimento aberto, e consiste de diversos componentes chamados de 'consolidações'. Visite a página de downloads para baixar os builds de desenvolvimentos disponíveis. Você pode navegar o código fonte com o
    OpenGrok.
  • Sistema Operacional OpenSolaris: Esta é uma distribuição binária desenvolvida pela comunidade com base do código fonte do OpenSolaris e apresenta as tecnologias mais recentemente desenvolvidas. A distribuição roda em arquiteturas SPARC e x86 em diversos ambientes (servidores, plataformas de armazenamento, desktops, laptops, ...). A utilização, modificação e redistribuição é gratuíra.
  • Sistema Operacional Solaris: Este é o sistema operacional enterprise level com suporte da Oracle. Futuras versões do Solaris serão baseadas em tecnologias do projeto OpenSolaris.

Gostaria de criar um grupo de usuários, como começo?

É simples, você precisa de uma lista de discussão e uma página na web. Você pode conseguir isto no domínio opensolaris.org, leia esta página com as regras e instruções, e envie sua proposta.

Caso sua proposta seja aprovada, será criada uma lista e um projeto dentro do portal opensolaris.org para seu grupo. A partir dai, é só divulgar a existência do grupo entre amigos e colegas, lembre-se de que um grupo é qualquer grupo de pessoas interessadas em discutir uma tecnologia. Não importa se são dois ou duzentos membros, o fundamental é adquirir e difundir conhecimento entre estes.

Criei um grupo de usuários de OpenSolaris, como o adiciono ao portal?

É simples, mande um email para a lista nacional de OpenSolaris com o nome do grupo, o endereço da lista de discussões e do website. Assim que possível, seus dados serão adicionados.
