About the Flask Architecture and SELinux

The Inevitability of Failure: The Flawed Assumption of Security in Modern Computing Environments argues that secure operating systems must be the basis of system security

The Flask Security Architecture: System Support for Diverse Security Policies describes the Flask security architecture, which is used by the FMAC project

In addition to these papers, visit the National Security Agency's SELinux Documentation page for more papers,
technical reports, and presentations about the security enhanced Linux (SELinux) project.

For an introduction to SELinux, see Getting Started with SE Linux HOWTO: the new SE Linux by Faye Coker.
