Using OpenSolaris

Distributionen, die auf dem OpenSolaris Quelltext basieren:
Solaris Express Community Edition – DVD: Download | CD: Download

 Solaris Express Community Edition ist Suns OpenSolaris Version für Entwickler (Codename "Nevada"). Es enthält die jeweils neuesten Entwicklungen von OpenSolaris, allerdings ohne jegliche kommerzielle Unterstützung, einschließlich anderer Technologien die nicht im OpenSolaris Quelltext verfügbar sind. Entwickler können den OpenSolaris Quelltext kompilieren und mit diesem Release als Basis ausprobieren. Solaris Express Community Edition wird jeden Freitag in der jeweils aktuellen Version zur Verfügung gestellt.

Solaris Express Developer Edition - Download | More Info

Solaris Express Developer Edition ist Suns offizielle Distribution basierend auf den OpenSolaris Bits. Sun bietet in begrenztem Rahmen Unterstützung für diese Version. Sie ist haupstsächlich für Solaris Kunden gedacht, die die jeweils neueste Technologie ausprobieren wollen, die eventuell in neue Solaris Versionen Einzug findet. Solaris Express wird jeweils monatlich in einer neuen Version zur Verfügung gestellt und ist als kostenloser Download verfügbar.

BeleniX – Download | More Info

 BeleniX ist eine *NIX Distribution die direkt auf OpenSolaris Quelltext basiert. Sie stellte zur Zeit eine sogenannte "Live CD" zur Verfügung (d.h. die Distribution kann direkt von CD gestartet werden und muß nicht erst installiert werden). Eventuell werden spätere Versionen auch konventionell auf die Festplatte installierbar sein.

marTux – Download | More Info

 marTux ist die erste OpenSolaris Distribution für SPARC Prozessoren (zur Zeit nur für die sun4u Architektur, sun4v ist in Planung).

NexentaOS – Download | More Info

 NexentaOS ist ein vollständig GNU Software basiertes freies und offenes Betriebssystem welches auf dem OpenSolaris Kernel und der Laufzeitumgebung aufbaut. NexentaOS integriert neben dem OpenSolaris (SunOS) Betriebssystemkern andere Open Source Software (OSS) Programme.

SchilliX – Download | More Info

 SchilliX ist eine OpenSolaris basierte UNIX Live CD und Distribution für x86, x64 und EM64T Mikroprozessor-Architekturen. Das zur Zeit aktuelle Release basiert auf Nevada Build 17 und späteren Versionen.

OpenSolaris verstehen

Das OpenSolaris Projekt hat ein Einsteiger Paket für Entwickler entworfen, die lernen wollen wie OpenSolaris installiert, gestartet und verwendet wird. Das Einsteiger Paket enthält einen Vergleich von OpenSolaris mit anderen Betriebssystemen.

OpenSolaris selber bauen

Das OpenSolaris Projekt besteht aus einer Reihe verschiedener Software Komponenten. Die größeren Komponenten werden auch als "Consolidation", bzw. Konsolidierungen bezeichnet. Die "Operating System/Networking" (ON) Konsolidierung bildet das Basissystem von OpenSolaris. Um OpenSolaris aus den Quelltexten zu bauen wird zunächst eine passende Distribution benötigt. Zur Zeit ist dies noch beschränkt auf das "Solaris Express Community Release" (siehe oben).

OpenSolaris Consolidations

Die Operating System/Networking (ON) Konsolidierung bildet die Basis für OpenSolaris.

Operating System/Networking (ON)

 ON enthält den Betriebssystemkern, Netzwerkdienste und Systembibliotheken und Kommandos.

Weitere OpenSolaris Konsolidierungen:

Application Server

 Der Sun Java System Application Server.

Developer Product Tools (DevPro)

 Die DevPro Konsolidierung enthält eine Vielzahl von Komponenten: Die mathematische Bibliothek, Eine Medien Bibliothek, SCCS, make und C++ Laufzeit Bibliotheken. Einige sind im Quelltext verfügbar, andere nur in Binärform.

Dokumentation

 Dokumentation für Entwickler und Administratoren. Viele dieser Bücher sind im Quelltext Format verfügbar.

Globalisierung

 Unterstützung für Internationalisierung und Lokalisierung der OpenSolaris Software.

Installation

 Installation und die entsprechenden Werkzeuge zum Erstellen von installierbaren Programmpaketen .

Java Desktop (JDS)

 Eine sichere und umfassende Umgebung für den Computer Arbeitsplatz.

Java Platform, Standard Edition (Java SE)

 Die Java Entwicklungsumgebung (JDK) bzw. Laufzeitumgebung (JRE) .

Manual Pages

 Quelltexte für die SunOS Referenz Handbücher.

Message Queue

 Die Sun Java System Message Queue.

Network Storage (NWS)

 Unterstützung für Netzwerkspeicher.

SFW (Solaris FreeWare)

 Open source Software, die mit OpenSolaris zusammen vertrieben wird.

SPARC Graphics Support

 SPARC Graphiktreiber in binärer Form.

Test

 OpenSolaris Testumgebung.

X Window System (X11)

 X11 Software.

Weitere OpenSolaris Konsolidierungen werden zu einem späteren Zeitpunkt veröffentlicht. Einzelheiten finden sich auf der Roadmap.

Hilfsmittel

Die Sun Studio 11 Compiler sind die empfohlenen Werkzeuge um den OpenSolaris Quelltext zu kompilieren. Einige Konsolidierungen arbeiten an der Unterstützung des GCC. Details finden sich auf den jeweiligen Download Seiten.
