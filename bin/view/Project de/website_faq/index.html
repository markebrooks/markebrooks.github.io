    Was ist der Zweck dieser Webseiten?
    OpenSolaris.org ist zentrale Stelle für alle, die zur Weiterentwicklung der OpenSolaris Quelltexte interessiert sind. Die OpenSolaris Community besteht sowohl aus professionellen Entwicklern als auch Anwendern, sowohl von innerhalb als auch außerhalb der Firma Sun Microsystems. Beiträge sind auf vielfältige Weise möglich, einschließlich dem Berichten von Fehlermeldungen, Schreiben von Dokumentation und nicht zuletzt Schreiben von Programmcode.
    Muss ich mich registrieren um eine Nachricht zu versenden?
    Ja. Gäste können die Diskussionsforen zwar lesen, aber nur registrierte Benutzer können selbst Nachrichten an die Foren senden.
    Wie abonniere Nachrichten in einem Forum und wie bestelle ich sie wider ab?
    Nach der erfolgreichen Registrierung können Foren durchsucht und mit einem Mausklick auf 'watch forum' - d.h. 'Forum beobachten' - abonniert werden. Das Mailsystem wird dann alle Nachrichten aus den ausgewählten Foren an die bei der Registrierung angegeben E-Mail Adresse schicken. Um den Empfang dieser Meldungen zu beenden reicht ein Mausklick auf 'Stop watching forum' - d.h. 'Aufhören dieses Forum zu beobachten'.
    Die Adresse, die bei der Registrierung eingetragen wurde, kann im Profil geändert werden. Das Profil wird durch einen Mausklick auf den Benutzernamen in der oberen rechten Ecke der Webseite geladen. (Ebenfalls zu diesem Thema: Wie ändere oder lösche ich Information in meinem Profil?)
    Ist es erlaubt dieselbe Meldung an mehrere Gruppen auf einmal zu versenden?
    Nein. Das sogenannte "Cross-Posting" sollte vermieden werden.
    Ist es erlaubt Stellenangebote zu versenden?
    Nein.
    Kann ich eine Nachricht wieder löschen?
    Nein. All Nachrichten, die an OpenSolaris.org versendet werden, werden archiviert. Informationen, die nicht zur Veröffentlichung bestimmt sind, dürfen auch nicht an die Diskussionsforen geschickt werden. Die Veröffentlichungspolitik der Webseite sieht vor die Archive unverändert zu lassen. Ausnahmen sind Spam, Viren, oder Quelltexte deren Rechte keine Veröffentlichung erlauben.
    Wenn eine Nachricht eine E-Mail Adresse als Teil einer Signatur enthält wird diese ebenfalls archiviert. Es werden keinerlei Maßnahmen ergriffen diese aus dem System zu entfernen.
    Kann ich Ankündigungen versenden?
    Unter Umständen. OpenSolaris.org verfügt sowohl über eine eigene Seite für Ankündungen bezüglich OpenSolaris im Allgemeinen, als auch über Seiten für die jeweiligen Communities. Ankündigungen, die für die gesamte Community interessant erscheinen, können an Derek Cicero (derek DOT cicero AT sun DOT com) oder Jim Grisanzio (jim DOT grisanzio AT sun DOT com) gesendet werden. Für eine Ankündigung innerhalb einer Community sollte der jeweilige Gruppenleiter kontaktiert werden.
    Wird meine persönliche Kontaktinformation auf der Seite veröffentlicht?
    Dies liegt im Ermessen des jeweiligen Benutzers. Beim erstellen eines Benutzerprofils wird sowohl ein Pseudonym zur Authentifizierung gegenüber dem System erstellt, als auch ein "wirklicher Name". Beide Angaben sind auf der OpenSolaris.org Webseit durchweg sichtbar um einen Benutzer zu identifizieren. Es ist nicht erforderlich seinen realen Namen anzugeben, auch für den wirklichen Namen kann ein beliebiges Pseudonym gewählt werden, z.B. "SolarisAnwender2000". Alle Nachrichten in den Foren verwenden den Benutzernamen, die E-Mail Adresse wird nicht angezeigt.
    Optional kann auch eine E-Mail Adresse im Benutzerprofil eingetragen werden.
    Wie ändere oder lösche ich Information in meinem Profil?
    Um an die Eintragungen im Profil zu gelangen reicht ein Mausklick auf den Benutzer- oder Realnamen in der rechten obereren Ecke auf jeder Webseite des OpenSolaris.org Portals. Von dieser Profilseite aus können beliebige Änderungen an den Feldern gemacht werden, mit Ausnahme des Benutzernamens.
    Um das Benutzerkonto zu löschen sollte Derek Cicero (derek DOT cicero AT sun DOT com) benachrichtigt werden.
    Welche anderen Quellen über Solaris befinden sich im Netz?
    Es gibt eine ganze Reihe von Webseiten rund um Solaris. Eine kleine Auswahl:
  • Solaris x86 Nachrichtenliste auf Yahoo http://groups.yahoo.com/group/solarisx86
  • BigAdmin auf Sun.com http://www.sun.com/bigadmin/
  • Solaris Gruppe im Usenet http://groups-beta.google.com/group/comp.unix.solaris?hl=en
    Eine ausführlichere Liste von Webseiten zum Thema Open/Solaris befindet sich auf der Seite mit Verweisen.
