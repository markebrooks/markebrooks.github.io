Lizenzbestimmungen

OpenSolaris.org enthält Verweise auf OpenSolaris Quelltexte, ausführbare Programme und Dokumentation die teils von OpenSolaris.org selbst und teils aus anderen Quellen heruntergeladen werden können. Es wird empfohlen vor dem Kopieren von Daten oder Programmen immer zuerst die anzuwendende Lizenz zu prüfen.

Diese Seite gibt einen Überblick über die Lizenzen, die auf OpenSolaris.org Anwendung finden.

Quelltext-Lizenzen

Die Mehrheit des OpenSolaris Quelltextes steht unter der  Common Development and Distribution License (CDDL) Version 1.0 (.txt). Die CDDL ist von der Open Source Initiative (OSI) als Open Source Lizenz anerkannt. Häufig gestellte Fragen bezüglich der CDDL sind hier beantwortet.

Einige Bestandteile des OpenSolaris Projektes basieren auf anderen Open Source Projekten. Diese Komponenten stehen weiterhin unter ihrer ursprünglichen Lizenz.

  • Perl in der ON Konsolidierung steht unter der  Perl Artistic License;
  • GNOME in der JDS Konsolidierung steht unter der GNU General Public License;
  • X11 in der X Konsolidierung steht unter der MIT License.

Der vollständige Text aller von der OSI anerkannten Lizenzen kann auf der OSI Webseite eingesehen werden.

Jede Konsolidierung bzw. jedes Projekt definiert die Lizenz für Beiträge selbst. Einzelheiten befinden sich auf den jeweiligen Projektseiten.

Lizenzen für nicht im Quelltext verfügbare Bestandteile

Die OpenSolaris Binary License findet bei Bestandteilen Anwendung, die auf nicht-offenem Quelltext beruhen oder die mehreren Lizenzen unterliegen. Einige dieser Bestandteile werden aus rein zweckdienlichen Gründen in Binärform zur Verfügung gestellt. Andere beruhen auf Quelltext, der nicht, oder noch nicht, als Open Source zur Verfügung steht. Diese Bestandteile werden in einem Archiv im TAR (Tape Archive) Format vertrieben. Das TAR Archiv enthält auch die anzuwendende Lizenz. Weitere Information befinden hier.

Die oben genannten Lizenzen sind nicht die einzigen. Zum Beispiel stehen einige Treiber unter einer BSD Lizenz.

Dokumentations Lizenz

Die OpenSolaris Dokumentation ist ebenfalls an Lizenzen gebunden

  • Die CDDL findet Anwendung bei Dokumentation die direkt zu einer Software gehört. Beispiele sind die Handbuchseiten (Man Pages) und die integrierte Hilfe.
  • Die Public Documentation License (PDL) v1.01 wird für Dokumentation verwendet, die separat von der Software ausgeliefert wird, zum Beispiel herkömmliche technische Anleitungen. Weitere Hinweise befinden sich hier.

Stehen Quelltexte unter anderen Lizenzen wird das wo immer möglich und sachdienlich in diesen Dateien vermerkt.
