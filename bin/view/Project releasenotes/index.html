NOTE: The Release Notes Project is no longer active. For information about Documentation, see the Documentation Community. For questions about opensolaris.org website infrastructure policies, see the Website User Guide or send mail to website-admin AT opensolaris DOT org.

Welcome to the Release Notes Project. 

The release notes for the OpenSolaris binary distribution have traditionally been hosted on opensolaris.org. This has provided an open forum for contributing to release notes material, but has also been a confusing place to publish the final product release notes for the binary distribution. Going forward, we want to provide a clearly identified community facing forum for collecting and discussing topics suitable for the release notes, no matter where the final release notes document is published. Hence, the idea to create a Release Notes project. The content supplied and vetted via these project pages will serve as raw data for the official release notes for the OpenSolaris binary distribution, independent of where those release notes are published.

Join the Project

There are different levels of participation. Choose the one that is right for you:

Participant

Project participants can leave comments on the project pages via the View > Comments link. To make yourself a Participant:

  1. Log in to the Auth system
  2. Click "Edit your account."
  3. Click "Collectives."
  4. In the Modify your collective relationships box, use the dropdown selections to make yourself a Participant of the Release Notes project. 

Developer

Project developers can leave comments on the project pages via the View > Comments link, edit pages, and commit to the source code repository. To become a Developer:

  1. Contact a Project Leader. Describe why you need Developer privileges.
