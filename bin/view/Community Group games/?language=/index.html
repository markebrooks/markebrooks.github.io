Games on OpenSolaris
The OpenSolaris games community is here to,

  • list games that can be played on OpenSolaris.
  • provide extra install instructions when needed.
  • host games that have been packaged for OpenSolaris.
  • host new games that OpenSolaris contributors have written.

How you can help
The following are ways people can help out,

  • Send OpenSolaris game info to the mailing list.
  • Package games for OpenSolaris, and submit them to blastwave, SFW or here.
  • Write instructions for installing a game on OpenSolaris.
  • Get involved writing new games!
    Packaging games is especially valuable, as they make for easy installations rather than wrestling with a compiler. Be careful to obey all licenses that come with the game (I'm sure that's a given). Games that have been on this site for a while and are widely tested, may be moved to blastwave/sunfreeware as their final destination.
    See the todo list for the current list of requests.
