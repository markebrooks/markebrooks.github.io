PowerPC port Tools

Cross toolchain for initial PowerPC port bring up. The tools are for Solaris x86 host.
GNU binutils 2.16 and GCC 4.0.0 are used during initial bootstrap phase.

NOTE: Currently the toolchain is 32 bit only. The great quest of creating biarch toolchain is still waiting for its champion. If you want to volunteer for the task speak up.

GNU binutils 2.16 and GCC 4.0.0 binary packages and patches against vanilla releases
