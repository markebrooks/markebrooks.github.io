Logical Domains

LDoms Overview

Logical Domains, or LDoms, technology allows a user to allocate a system's various resources, such as CPU, memory, and IO into logical groupings or domains. This enables one to create multiple discrete systems, each with their own operating system, resources, and identity within a single physical computer system.  The official product name is Oracle VM Server for SPARC.
