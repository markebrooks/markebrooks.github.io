Web サイトに関する FAQ

  • このサイトの目的は何ですか。
  • メッセージを投稿する場合にはサイトへの登録が必要ですか。
  • フォーラムへの参加申し込みや取り消しはどうすればできますか。
  • 複数のメーリング・リストに重複して投稿してもいいですか。
  • 求人情報を投稿してもいいですか。
  • メッセージを削除できますか。
  • アナウンスを発表できますか。
  • 個人的な連絡先情報はサイト上で公表されるのですか。
  • 自分のプロフィール情報の編集や削除はどうすればできますか。
  • Solaris オペレーティング・システム (OS) 用のオンライン・リソースには、ほかにどのようなものがありますか。
    このサイトの目的は何ですか。
    opensolaris.org は、OpenSolaris コード・ベースの将来の開発を共同で行うことに興味をもつ人々にとってのオンライン・ホームです。OpenSolaris コミュニティは、Sun の内部や外部からのプロの開発者やエンドユーザで構成されます。コミュニティの参加者は、バグ・レポートの提出や、ドキュメントの作成、コードの開発など、いろいろな方法で参加できます。
    メッセージを投稿する場合にはサイトへの登録が必要ですか。
    はい、必要です。ディスカッション・フォーラムの閲覧だけならゲストとしてできますが、メッセージを投稿するためには、ユーザ・アカウントを作成し、eメール・アドレスの承認を受ける必要があります。
    フォーラムへの参加申し込みや取り消しはどうすればできますか。
    アカウントを作成したら、フォーラムをブラウズし、興味のあるトピック (複数可) に対して「Watch Forum」をクリックします。その後、選択したトピックに関するすべてのメッセージが、登録した eメール・アドレスに送信されます。特定のトピックについての eメールの受信を停止する場合は、「Stop Watching Forum」をクリックします。
    eメール・アドレスを変更する場合は、画面の右上隅にある自分の Username をクリックすると、プロフィール・ページが表示されます。(下記の「自分のプロフィール情報の編集や削除はどうすればできますか。」もご覧ください。)
    複数のメーリング・リストに重複して投稿してもいいですか。
    いいえ、できません。複数のメーリング・リストに重複して投稿することは避けるべきです。
    求人情報を投稿してもいいですか。
    いいえ、いけません。
    メッセージを削除できますか。
    いいえ、できません。opensolaris.org メーリング・リストに送信されたすべてのメッセージはアーカイブされます。したがって、公表したくない情報は、フォーラムに送信しないでください。サイトの方針として、アーカイブの情報は、SPAM やウイルス、所有権のある (proprietary) コードなどを除き、絶対に編集を行いません。
    さらに、署名ファイルに eメール・アドレスが含まれていると、それもアーカイブに格納されます。このアドレスをシステムから削除することは特にしません。
    アナウンスを発表できますか。
    可能だと思います。このサイトには、OpenSolaris 全体に関する発表のページと、個別のコミュニティに関する発表のページがあります。発表が OpenSolaris コミュニティ全体に関係すると思われる場合は、それを Derek Cicero (derek dot cicero at sun dot com) か Jim Grisanzio (jim dot grisanzio at sun dot com) に送信してください。特定のコミュニティ向けに発表を投稿する場合は、適切なコミュニティ・リーダーに連絡してください。投稿したい発表が承認されると、サイトまたはコミュニティのリーダーより通知があります。
    個人的な連絡先情報はサイト上で公表されるのですか。
    それは本人次第です。ユーザ・アカウントを作成するときに、「Username」と「Real Name」を指定する必要があります。これらのフィールドの情報は、自分専用のページを含めシステムのすべてのページに識別情報として表示されます。実名を表示したくない場合は、「Real Name」として「SolarisLover57」などハンドル名を使用できます。さらに、フォーラムへの投稿には Username が使用されるので、投稿者の eメール・アドレスが自動的に掲載されることはありません。したがって、必要なら匿名のまま利用できるだけでなく、スパム・ハーベスタ (スパーム・メールに使用されるメール・アドレス収集プログラム) の被害を回避できます。
    また、eメール・アドレスなど個人情報を自分専用のページに掲載することもできます。
    自分のプロフィール情報の編集や削除はどうすればできますか。
    プロフィール情報を編集する場合は、各ページの右上隅にある Username ＆ Real Name (ハイパーリンク) をクリックします。これによって表示される自分のプロフィール・ページでは、Username フィールドを除く任意のフィールドを必要に応じて変更できます。
    アカウントを削除する場合は、Derek Cicero (derek dot cicero at sun dot com) に連絡してください。
    Solaris オペレーティング・システム (OS) 用のオンライン・リソースには、ほかにどのようなものがありますか。
    現在、いくつかの Solaris コミュニティとドキュメント・リポジトリで Solaris OS の情報が提供されており、Solaris に関連するディスカッション・フォーラムが運営されています。次のサイトをご覧ください。
  • Yahoo の Solaris x86 メーリング・リスト http://groups.yahoo.com/group/solarisx86
  • Sun.com の BigAdmin http://www.sun.com/bigadmin/
  • Solaris Usenet ニュースグループ http://groups-beta.google.com/group/comp.unix.solaris?hl=en
    Open/Solaris 関連のその他のサイトについては、リンク・ページをご覧ください。
