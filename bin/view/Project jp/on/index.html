Nevada は、Solaris 10 の次の Solaris リリースに Sun が与えたコードネームです。OpenSolaris と Nevada は、同じソース・ベースからビルドされます。

このコミュニティは、Nevada と OpenSolaris に取り組んでいる OS/Net (ON) 開発者を中心としています。現在、このコンテンツはまだ十分ではありません。Sun が社内の ON Nevada Web リソースを opensolaris.org に移行させるにしたがって、コンテンツを追加し再編成する予定です。

 現在はまだ、社内サイトベースで作業中であり、機能しないリンクが多数あります。
