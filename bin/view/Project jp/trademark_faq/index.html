OpenSolaris の商標/ロゴの使用に関する FAQ

  • OpenSolaris は Sun の商標ですか。
  • OpenSolaris の商標はどのような使い方が許されるのですか。
  • 派生的な著作物やディストリビューションに OpenSolaris の名前を使用できますか。
  • OpenSolaris のファン・ボタンやバナーは何のためにあるのですか。
  • OpenSolaris のファン・ボタンやバナーはどこで使用できますか。
  • ファン・ボタンやバナーの使用には登録が必要ですか。
  • OpenSolaris のファン・ボタンやバナーの使用を管理する商標ライセンスはなぜ必要なのですか。
  • ボタンやバナーの周りにスペースを設けるという指針はなぜ必要なのですか。
  • OpenSolaris の現状のファン・ボタンやバナーを変更することはできますか。
  • 開発中の派生的な著作物が OpenSolaris プロジェクトと関係があることを公表したいのですが、そのためのプログラムはいつ発表されますか。
    OpenSolaris は Sun の商標ですか。
    はい、OpenSolaris は Sun Microsystems, Inc. の商標です。この FAQ には、OpenSolaris 商標の使用に関する主要な情報が含まれています。Sun の商標の使用に関するそのほかの情報については、Sun 商標ポリシーのページをご覧ください。
    OpenSolaris の商標はどのような使い方が許されるのですか。
    まず第一に、OpenSolaris のファン・ボタンとバナーがすでに使用可能になっています。このファン・ボタンやバナーを使用すれば、OpenSolaris コミュニティや技術を支持しサポートしていることを自身の Web サイトで簡単に明示することができます。ファン・ボタンは簡単に入手できるだけでなく、登録も必要ありません。ただし、「使用指針」や、ファン・ボタンやバナーの使用を管理する「商標ライセンス」はあります。
    (読者のみなさまへ。OpenSolaris のファン・ボタンとバナーを気に入っていただけましたか (われわれは気に入っていますが)。これらのボタンやバナーを使って、OpenSolaris プロジェクトが広く認知されるようにしたいと思います。)
    第二に、OpenSolaris 商標の公正な使用はもちろん認められています。たとえば、文書やブログなどで OpenSolaris プロジェクトに言及したり、意見を述べたりする場合です。さらに、これから増加する OpenSolaris ユーザ・グループがその名前の一部に OpenSolaris の商標を使用することもできます。ただし、ユーザ・グループに対する Sun の商標ポリシーに準拠していなければなりません。
    第三に、派生的な著作物に OpenSolaris 技術が含まれていることを公表できるプログラムを現在、作成しています。さらに、Web サイトやシステムで OpenSolaris 技術が動作していることを公表するためのボタンを開発する予定です。引き続きご注目ください。
    派生的な著作物やディストリビューションに OpenSolaris の名前を使用できますか。
    いいえ、OpenSolaris ソース・コードを利用してソフトウェアやディストリビューションを作成する場合、その製品名の一部に OpenSolaris の商標を直接使用することはできません。たとえば、「My OpenSolaris」は使用できません。
    現在、派生的な著作物に OpenSolaris 技術が含まれていることを公表できるプログラムを作成しています。将来、このプログラムが発表されれば、ボタンやバナーを使って、派生的な著作物と OpenSolaris 技術の関連性を明示できるようになります。たとえば、キャッチフレーズとして「Based on OpenSolaris」を採用することにした場合、ユーザは製品を「Kamal's Operating System - Based on OpenSolaris」と呼ぶことができます。ただし、もう少々お待ちください。
    OpenSolaris のファン・ボタンやバナーは何のためにあるのですか。
    ファン・ボタンやバナーのデザインはなかなかいいと思いませんか。自分の Web サイトにこれらのボタンやバナーを載せることによって、プロジェクトの認知度を高め、サポートしていることを明示したいと思っています。OpenSolaris ボタンをご覧ください。
    OpenSolaris のファン・ボタンやバナーはどこで使用できますか。
    Web サイト、ブログ・ページ、インターネットで使用できます。試してみてください。
    ファン・ボタンやバナーの使用には登録が必要ですか。
    いいえ、登録は必要ありません。OpenSolaris のファン・ボタンやバナーを入手したい場合は、ボタンとバナーのページを開き、使用したいイメージをクリックしてください。ポップアップ・ウィンドウに HTML コードが表示されますので、それを Web ページに貼り付ければボタンを表示できます。ボタン・ダウンロード・ページには、使用上の指針や、「OpenSolaris ファン・ボタンおよびバナー商標」ライセンスの説明があります。
    OpenSolaris のファン・ボタンやバナーの使用を管理する商標ライセンスはなぜ必要なのですか。
    商標を専門とする弁護士の意見に基づき、ファン・ボタンやバナーとともに「商標ライセンス」を明示する必要がありますが、これは、世界的な商標法規の条項を遵守するためです。とはいえ、これらのファン・ボタンは簡単に入手できるようになっていますので、ユーザは、登録や書類手続きなしに OpenSolaris プロジェクトを支持していることを明示できます。
    指針や、商標ライセンスの諸条項に従ってください。それによって、全員で OpenSolaris コミュニティの認知度を高めながら、同時に、OpenSolaris 商標を保護することができ、すべての人が望む価値や品質のレベルを維持することができます。
    ボタンやバナーの周りにスペースを設けるという指針はなぜ必要なのですか。
    2 つのロゴが接近しすぎると、それらの製品やプロジェクトが互いに無関係で独立しているにもかかわらず、何らかの関係があるものと誤解されるおそれがあります。したがって、OpenSolaris のファン・ボタンやバナーは、ほかのロゴやグラフィックスとビジュアル的に明確に区別され、ほかのデザイン要素と重ならない必要があります。(ボタン・ページの HTML ではこの点が考慮されています。)
    OpenSolaris の現状のファン・ボタンやバナーを変更することはできますか。
    いいえ、できません。ボタンやバナーは現状のまま使用してください。世界的な商標の要件や慣行を遵守するためには、ボタンやバナーを現状のまま使用する必要があります。しかし、改善のためのアイディアは歓迎します。opensolaris dash mktg at opensolaris dot org の OpenSolaris コミュニティ・マーケティング・リストを使ってさまざまな提案に対して議論したいと思います。
    開発中の派生的な著作物が OpenSolaris プロジェクトと関係があることを公表したいのですが、そのためのプログラムはいつ発表されますか。
    OpenSolaris 技術が組み込まれ、派生した著作物に関する商標使用プログラムを作成する予定です。このプログラムには、商標ライセンスとアートワークが含まれています。もう少しお待ちください。
