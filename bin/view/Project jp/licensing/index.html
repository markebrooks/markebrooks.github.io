ライセンス

ライセンス情報

ソースコード、実行ファイルおよびドキュメントは opensolaris.org からダウンロードできます。 このサイトは、他のサーバーからダウンロードできる情報の ポインターもあります。 このようなデータは、ライセンスされています。これらのライセンスについては ダウンロードに先だってにお読みください。

このページでは、opensolaris.org で使用されている一般的なライセンスの ほとんどが概説されています。ダウンロードを実行する前に、 すべての情報をよくお読みください。

ソースコードライセンス

 ほとんどの OpenSolaris のソースコードは次のライセンスのもとにリリースされています。  Common Development and Distribution License (CDDL) Version 1.0 (cddllicense.txt) (*Chinese Translation*)。 CDDL Open Source Initiative (OSI) はオープンソースのための承認されたライセンスです。 CDDL の詳細については、CDDL FAQ をご参照ください。

 OpenSolaris プロジェクトは、既存のオープンソースプロジェクトに基づいた多くのコンポーネントを含んでいます。 これらのコンポーネントは、現在のライセンスのもと、将来も使用できます。たとえば、

  • ON consolidation の Perl は  The Perl Artistic License にもとづいてライセンスされています。
  • JDS consolidation の GNOME は GNU General Public Licenseにもとづいてライセンスされています。
  • X consolidation の X11 は MIT license にもとづいてライセンスされています。

 OSI で承認されたライセンスの全文は、OSI website で参照できます。

 opensolaris.org の各プロジェクトやグループは貢献された情報に対して使用するライセンスを定義しています。 詳細については、各プロジェクト情報を参照してください。

バイナリライセンス

OpenSolaris バイナリライセンス は、著作権付きのソースや 複数のライセンスが適用されているソースからビルドされたバイナリに対して使用されます。 いくつかのバイナリは、利用可能な状態で提供されていますが、オープンソースとしてリリースされていない、 あるいは、まだオープンソースとしてのリリースの準備が整っていないソースからビルドされたバイナリもあります。 これらのバイナリについては、ダウンロード可能な各 tar (圧縮) ファイルに、 OpenSolaris バイナリライセンスの全文を含むファイルがひとつずつ入っています。 詳細については OpenSolaris Binary License FAQ を参照してください。

 ダウンロードが可能なバイナリの中にはその他のライセンスも適用されている場合がありますので、ご注意ください。 たとえば、BSD 形式のライセンスのもとに提供されているいくつかのドライバーのバイナリが含まれている可能性があります。 また、利用可能なバイナリであっても、CDDL を使ってライセンスされている ソースからビルドされた可能性もあります。従って、このようなバイナリは CDDL が適用されます。

ドキュメントのライセンス

OpenSolaris のドキュメントソースは、次のふたつのうちのひとつのライセンスの下に提供されます。

  • CDDL はソフトウェアのパッケージ構造の一部として デリバーされるドキュメントのソースをライセンスするために使用されます。 この例は、manpage やオンラインヘルプに含まれています。
  • 公共ドキュメントライセンス (PDL: Public Documentation License) v1.01 は、 そのドキュメントが説明しているソフトウェアと別にデリバーされるドキュメントソースをライセンスするために使用されます。 この例は、従来の技術マニュアルに含まれます。このライセンスの詳細については PDL FAQ を参照してください。

サードパーティーやオープンソースのライセンスは、必要に応じて、ドキュメントソースに含まれてます。
