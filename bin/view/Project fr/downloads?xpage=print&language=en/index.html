Centre de Téléchargement OpenSolaris

L'essentiel du code d'OpenSolaris est disponible sous la Licence Commune de Développement et de Distribution (CDDL) Version 1.0.
Le source basé sur des projects Open Source existants continue d'être disponible sous ses licences actuelles. Quelques composants binaires sont couvert par la Licence Binaire OpenSolaris et d'autres le sont sous d'autres licences Open Source. Les pages de téléchargement correspondantes fournissent les informations sur la licence associée à chaque composant.

Utiliser OpenSolaris

Si vous désirez utiliser une distribution basée sur OpenSolaris, vous pouvez choisir une de ces options:
OpenSolaris 2008.05 – Télécharger

 OpenSolaris 2008.05 est un unique CD à la fois Live et installation: un système d'exploitation de base, noyau, bibliothèques systèmes, un environnement de bureau et un système de gestion des paquets.

Solaris Express Community Edition («Édition Communauté») – DVD: Télécharger | CD: Télécharger

 Solaris Express Community Edition est la version binaire de Sun pour les développeurs (nom de code «Nevada»). Elle contient les tous derniers morceaux pas encore supportés d'OpenSolaris, ainsi que des technologies supplémentaires qui n'ont pas encore été livrées dans la base du code source d'OpenSolaris. Les développeurs peuvent compiler le source d'OpenSolaris en utilisant cette version comme système de base. Elle est mise à jour un vendredi sur deux.

Solaris Express Developer Edition («Édition Développeur») – Télécharger | Plus d'informations

Solaris Express Developer Edition est la version officielle de Sun des éléments d'OpenSolaris ainsi que des technologies supplémentaires qui n'ont pas encore été livrées dans la base du code source d'OpenSolaris. Pour plus de détails, en particulier concernant les possibilités de support, consultez le site de l'Édition Développeur. Cette distribution est gratuite. L'Édition Développeur est livrée tous les 3 à 4 mois, et remplace l'édition mensuelle de Solaris Express.

BeleniX – Télécharger | Plus d'informations

 BeleniX est une distribution *NIX assemblée sur le code source d'OpenSolaris. C'est actuellement une distribution Live CD, mais elle vise à devenir une distribution complète qui pourra être installée sur disque dur.

MartUX mBE – Télécharger | Plus d'informations

 MartUX BlastwaRe Edition est la première distribution OpenSolaris hors SXCR disponible à la fois sur SPARC et sur x86/x64. Elle est actuellement disponible sur LiveDVD, et inclut des tonnes de logiciels utiles dans /opt/csw.

NexentaOS – Télécharger | Plus d'informations

 NexentaOS est un système d'exploitation Open Source complet basé sur GNU, construit sur le noyau et les programmes OpenSolaris. NexentaOS intègre OpenSolaris (le noyau SunOSà) et des applications logicielles Open Source (OSS), une fondation qui combine le meilleur des deux mondes.

SchilliX – Télécharger | Plus d'informations

 SchilliX est une distribution UNIX en Live CD, basée sur OpenSolaris, pour les architectures x86, x64 et EMT64. La version actuelle utilise Nevada Build 17 et au delà.

Apprendre OpenSolaris

Le projet OpenSolaris a créé un Kit de Démarrage pour les développeurs qui veulent apprendre à installer, démarrer et exécuter OpenSolaris. Le Kit de Démarrage inclut une comparaison d'OpenSolaris avec d'autres sytèmes d'exploitation.

Compiler OpenSolaris

Beaucoup de composants logiciels différents font partie du projet OpenSolaris, les composants essentiels sont connus sous le nom de «consolidations». La consolidation Système d'Exploitation/Réseau (Operating System/Networking - ON) constitue le support système de base pour OpenSolaris. Pour compiler ce dernier à partir des sources, vous devez d'abord installer une distribution OpenSolaris adaptée, ce qui est uniquement Solaris Express Community Release (voir plus haut).

Les Consolidations OpenSolaris

Beaucoup d'éléments logiciels différents composent le projet OpenSolaris. Les composants essentiels sont connus sous le nom de «consolidations». La consolidation Système d'Exploitation/Réseau (Operating System/Networking - ON) constitue le support système de base pour OpenSolaris.

Système d'Exploitation/Réseau (Operating System/Networking - ON)

 ON inclut le noyau, les services réseau, les bibliothèques et commandes système.

Autres consolidations OpenSolaris:

Serveur d'Application (Application Server)

 Sun Java System Application Server.

Les Outils de Dévelopement Produit (Developer Product Tools - DevPro)

 La consolidation DevPro a plusieurs composants disponibles: la bibliothèque système mathématique, la bibliothèque multimédia, la bibliothèque «microtasking», SCCS et make, ainsi que les bibliothèques d'exécution C++. Certains sont disponibles sous forme source, et d'autres, de binaires.

Documentation (Docs)

 La documentation technique de développement et d'administration. Le code source de plusieurs livres est disponible. Voyez la page de téléchargement pour les détails.

Support de Globalisation (G11N)

 Support pour l'internationalization et la localisation.

Support de l'Installation (Install)

 Support de l'installation et des outils de création de paquets.

Java Desktop (JDS)

 Un environnement de bureau pour entreprise, sécurisé et complet.

Plate-forme Java, Édition Standard (Java SE)

 Les binaires du Kit de Développement Java (JDK) et de l'Environnement d'Exécution Java (JRE) sont disponibles.

Pages de Manuel

 Le code source des Pages de Manuel de Référence SunOS.

Message Queue

 Le Sun Java System Message Queue. Un fichier zip Open Source est disponible.

Stockage Réseau (Network Storage - NWS)

 Support de périphériques de stockage réseau.

SFW (Solaris FreeWare)

 Les logiciels Open Source intégrés dans Solaris/OpenSolaris.

Support Graphique pour SPARC

 La consolidation graphique SPARC contient des pilotes disponibles sous forme binaire.

Test

 Les Outils de Test et les Suites de Test OpenSolaris.

Le Système X Window (X11)

 Les logiciels X11.

D'autres consolidations OpenSolaris seront disponibles plus tard. Voyez la feuille de route pour plus d'informations.

Outils

Il est recommandé d'utiliser les compilateurs Sun Studio 11 pour compiler le source d'OpenSolaris. Certaines consolidations travaillent au support de GCC en plus de Sun Studio. Les pages de téléchargement de chacune donnent plus de détails.
