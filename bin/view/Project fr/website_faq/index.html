    But du site opensolaris.org
    opensolaris.org est la ressource en ligne pour les personnes souhaitant collaborer au développement d'OpenSolaris. La communauté est composée de développeurs professionnels aussi bien que d'utilisateurs, de Sun et de l'extérieur. Vous pouvez participer de nombreuses manières, comme par exemple rapporter des bogues, écrire de la documentation, ou bien encore participer au développement.
    Est-il nécessaire de s'enregistrer pour poster un message?
    Oui. Vous pouvez librement parcourir les forums de discussions en tant qu'invité, mais pour poster un message, vous devez créer un compte utilisateur avec une adresse e-mail valide.
    S'abonner/se désabonner à un forum
    Une fois votre compte créé, vous pouvez parcourir les forums et cliquer sur «Suivre un forum» («Watch forum») pour un sujet que vous désirez suivre. Le moteur vous adressera alors tous les nouveaux messages à l'adresse que vous avez fournie lors de la création de votre compte. Pour ne plus recevoir de message, cliquez sur «Arrêt du suivi du forum» («Stop watching forum»).
    Si vous souhaitez modifier l'adresse de réception, cliquez sur votre Nom d'utilisateur dans le coin supérieur droit du site pour mettre votre profil à jour. (Voir également «Édition ou suppression de mon profil»)
    Comment signaler des abus
    Pour signaler un message qui enfreint les Règles d'Utilisation, tel que du spam, envoyez un mél contenant un lien vers le message à [help AT opensolaris POINT org].
    Si vous pensez que votre compte a été compromis, envoyez un mél contenant votre nom de connexion à [help AT opensolaris POINT org].
    Un membre de notre équipe vous répondra aussi rapidement que possible.
    À propos du multipostage (envoi simultané sur plusieurs forums)
    Le multipostage n'est pas autorisé.
    Puis-je envoyer une offre d'emploi?
    Non, cela est interdit.
    Puis-je supprimer un message?
    Non. Tous les messages envoyés sur les listes de diffusion d'opensolaris.org sont archivés. N'envoyez aucune information sur un forum que vous ne voulez pas rendre publique. La politique de notre site est de ne jamais éditer les archives sauf pour le spam, les virus, ou l'envoi de code propriétaire.
    De plus, si vous choisissez de mettre votre adresse e-mail dans votre fichier signature, elle sera incluse dans les archives, et nous ne ferons aucune tentative pour la supprimer du système.
    Puis-je poster des annonces?
    Peut-être. Le site contient une page pour les annonces OpenSolaris en général, comme pour chaque communauté. Si vous avez une annonce qui peut intéresser toute la communauté OpenSolaris, merci de l'envoyer à Derek Cicero (derek DOT cicero AT sun DOT com) ou Jim Grisanzio (jim DOT grisanzio AT sun DOT com). Pour poster une annonce au sein d'une communauté, contactez le leader de celle-ci. Il vous préviendra si l'annonce est approuvée.
    Est-ce que mes informations personnelles seront rendues publiques sur le site?
    C'est comme vous voulez. Quand vous créez votre compte utilisateur, vous choissisez un Nom d'utilisateur («Username») et un «Vrai Nom» («Real Name»). Ces informations seront montrées dans le système, en particulier sur votre page personnelle, pour vous identifier. Si vous ne souhaitez pas montrer votre vrai nom, vous pouvez choisir un pseudo en tant que «Vrai Nom», par exemple «SolarisLover57». Tous les posts sur les forums utiliseront votre Nom d'utilisateur, votre adresse e-mail ne sera donc jamais publiée. Cela vous permet de rester anonyme si vous le désirez (et d'éviter que votre adresse soit récupérée par les spammeurs).
    Vous avez aussi la possibilité de publier vos informations personnelles, comme l'adresse e-mail, sur votre page personnelle.
    Édition ou suppression de mon profil
    Pour modifier les informations de votre profil, cliquez sur votre Nom d'utilisateur dans le coin supérieur droit de chaque page. Vous pourrez modifier l'ensemble des informations à l'exception de votre nom d'utilisateur.
    Si vous souhaitez supprimer votre compte, veuillez contacter Derek Ciceco (derek DOT cicero AT sun DOT com).
    Quelles sont les autres ressources en ligne pour le Système d'Exploitation Solaris?
    Il y a actuellement plusieurs communautés Solaris, comme des sites qui contiennent des documents d'information sur le SE Solaris, et qui hébergent des forums de discussion sur Solaris. Il y a en particulier:
  • (anglophone) La liste Solaris x86 sur Yahoo http://groups.yahoo.com/group/solarisx86
  • (anglophone) BigAdmin sur Sun.com http://www.sun.com/bigadmin/
  • (anglophone) Les groupes de discussion Usenet de Solaris http://groups-beta.google.com/group/comp.unix.solaris?hl=en
  • (francophone) Les groupes de discussion Usenet de Solaris http://groups-beta.google.com/group/fr.comp.sys.sun?hl=en
    Pour une liste plus complète des sites Open/Solaris, voyez la page des liens.
