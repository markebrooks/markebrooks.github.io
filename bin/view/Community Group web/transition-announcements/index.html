Announcements

Website Development Announcements

NOTE: This page is now inactive. Website development projects and public announcements from 2011 and beyond are documented in the public roadmap. For questions about the website, see the Site User Guide or send mail to  website-admin AT opensolaris DOT org.

The documents linked below contain announcements from website development projects for opensolaris.org: plan development, source and binary releases, testing, open evaluation processes, conference calls, implementation details, and operations. More discussions about the website transition took place on the lists since 2007, but these are the major announcements for the overall project. Where links do not appear, either documents have been updated/replaced or the communication took place on a venue that does not offer links (either direct email to users or to a private list). In those cases, we merely document that the communication took place for transparency reasons, but please note that nothing confidential was discussed in those communications.
