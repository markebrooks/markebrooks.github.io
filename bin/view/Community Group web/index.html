Restructuring opensolaris.org

The current opensolaris.org website represents an entirely new platform from when the OpenSolaris Project started back in 2005. Planning to redesign and rebuild opensolaris.org began in late 2007. Details outlining the strategy for replacing the old website can be found at Restructuring the opensolaris.org infrastructure. Throughout 2009 the website restructuring project moved into the implementation phase with two major transitions to new infrastructure:

  • Phase 1, August 3, 2009: Transition to a new membership management application at auth.opensolaris.org.
  • Phase 2, October 26, 2009. Transition to new site at markebrooks.github.io with XWiki for content management.

See the following documents and sections outlining the project management details of the transition:

  • Website Transition FAQ
  • User Action FAQ
  • Site Feature Mappings
  • Content Preparation Instructions
  • Data Migration Strategy
  • Website Conference Calls

For schedules of ongoing website development operations beyond Phases 1 & 2 (as well as an archive since 2005), see the roadmap section. For all website restructuring and transition announcements from 2007 to present, see the announcements page. Send questions to website-discuss.

Bugs & RFEs

Post bugs and RFEs to Bugzilla: http://defect.opensolaris.org/bz/

  • For Auth, follow this path: Enter a new bug report -> Development -> Website (Component = site-auth-webapp)
  • For XWiki, follow this path: Enter a new bug report -> Development -> Website (Component = site-xwiki)

Contacts

Sun's website development, transition, and infrastructure teams can be found here:

  • For general community discussions about the website, post to website-discuss AT opensolaris DOT org. Subscribe here.
  • For website user account issues, post to website-admin AT opensolaris DOT org.
  • For implementation interactions concerning collective life cycle issues, post to website-admin AT opensolaris DOT org.
