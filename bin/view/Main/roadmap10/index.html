Roadmap 2010

Archive: 2011 | 2010 | 2009 | 2005-2008 | Consolidations | Website Announcements

JAN-MAR 2010 APR-JUN 2010 JUL-SEP 2010 OCT-DEC 2010
  • auth.os.org
    • Update: Added back end support for blog feeds, automatic changes from Core Contributor to Contributor status, and better integration with subsites.
      • 1/13/10
    • Update: Second edition of auth info screens showing all collective relationships and all user roles. This update also includes eight new localizations (for a total of 25 languages) and a separate web service to provide graphical elements (headers, footers, etc) across all os.org sites.
      • 2/5/10.
  • hub.os.org
    • Flash Support: Migrated files and documented XWiki macro for embedded media.
      • 1/7/10
    • Upgrade: Updated XWiki application from v1.8 to v2.1.1.
      • 1/21/10
  • poll.os.org
    • Rebuilt, deployed, and tested an updated voting application. Integrated poll with auth and provided new web front end for voting.
      • 2/22/10
    • Managed the poll infrastructure during election and calculated and published results.
      • 3/23/10
  • Miscellaneous
    • static.os.org deployed for ON Flag Days.
      • 2/20/10
  • repo.os.org
    • Rebuild the Source Code Management application with a new web console front-end interface and a new back-end infrastructure. Also, integrating the application with a header/footer web service and localizing it into 25 languages.
      • Target: Work completed during the entire quarter.
  • Data Center
    • Major Data Center consolidation due to the Oracle acquisition of Sun. This work affects the website development and deployment schedules on this roadmap, but website services will be maintained during the transition.
      • Target: Entire quarter
    • Major systems moved. Additional work ongoing.
      • 5/27/10
  • stage.os.org
    • Planned decommission.
      • 4/30/10
  • Localization
    • Updated auth, repo, chrome applications. 25 languages.
      • 5/28/10
  • hub.os.org
    • Enabled attachments to be displayed by default on XWiki pages.
      • 4/9/10
  • defect.os.org
    • Added SSL for authenticated sessions.
      • 4/21/10
  • bugs.os.org
    • Enabled subscriptions to bug report interests.
      • 5/24/10
  • repo.os.org
    • Deployed new SCM Console web front end.
      • 5/28/10
  • defect.os.org
    • Upgraded to Bugzilla v3.6.2
      • 08/26/10
  • repo.os.org
    • Deployed new back end SCM infrastructure and updates to new SCM Console.
      • 09/21/10
  • auth.os.org
    • Updated auth with latest bug fixes, added automatic checking of commits to project repositories to verify committer agreements, removed inactive accounts.
      • 09/21/10
  • hub.os.org
    • Deploy new XWiki skin and auth/hub integration code to update the app's look/feel, improve performance, fix style sheet bugs, fix watch lists, and relocate subsites menu out of the left nav.
      • 10/04/10
  • OpenGrok
    • Update to latest version.
      • 12/1/10
  • defect.os.org
    • Update Bugzilla to v3.6.3
      • 12/29/10
