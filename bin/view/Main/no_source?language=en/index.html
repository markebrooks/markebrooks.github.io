Source Not Released

OpenSolaris Components Currently Not Released as Source

This page contains information, available by consolidation, about technology components that are not available in source form.  When possible, re-distributable binaries are made available when source can not be provided.  If such binaries are available for the technology component listed, a "B" in parentheses will be included with the technology component name.

  • Globalization
  • Install
  • Java Desktop (JDS)
  • Manual Pages
  • Open HA Cluster
  • Operating System/Networking (ON)
  • SPARC Graphics Support
  • X Window System

Globalization

Components that we plan to release as source in the future

  • As many Japanese man pages and roff data files as possible

Components for which there are currently no plans to release source

  • Fonts licensed from non-open-source providers
  • ATOK Japanese language engine
  • Wnn Japanese language engine
  • CID files
  • WuBi Simplified Chinese input method
  • Array Traditional Chinese input method
  • Boshiamy Traditional Chinese input method
  • DaYi Traditional Chinese input method
  • Thai dictionary file
  • Korean language engine

Install

Components for which there are currently no plans to release source

  • Live Upgrade
  • Flash install code

Java Desktop (JDS)

Components that we plan to release as source in the future

  • Trusted Extensions for JDS

Components for which there are currently no plans to release source

  • Acroread
  • Flash
  • RealPlayer

  • Support for Veritas File System (VxFS)

Components that we plan to release as source in the future

  • smbus_ara driver (B) (Daktari platform support)
  • SunFire V240 platmod driver (B) (SunFire platform support)
  • SunFire V250 platmod driver (B) (SunFire platform support)
  • SunFire V440 platmod driver (B) (SunFire platform support)
  • UltraEnterprise platmod driver (B) (UltraEnterprise platform support)

Components for which plans are not currently set

  • bmc driver (Baseboard management controller)
  • SunFire V240 ntwdt driver (B) (Netra-based application watchdog timer driver)

  • iprb driver (B) (Intel Pro1000/B Fast Ethernet driver)

  • m1535ppm driver (B) (Acer ALI1535D and 1535D+ PCI PMU device driver)
  • mi2cv driver (B) (Nexus driver for Mentor Graphics MI2CV I2C controller)
  • mpt driver (B) (SCSI HBA driver that supports the LSI 53C1030 SCSI chip)
  • n2cp driver (B) (Niagara crypto driver)
  • ncp driver (B) (Niagara crypto driver)
  • ncrs driver (B) (SCSI HBA driver)
  • pcn driver (AMD PCnet Ethernet controller device driver)
  • pcser driver (B) (PCMCIA serial card device driver)
  • scfd driver (B) (OPL platform driver)
  • scsi_vhci driver (B) (SCSI virtual host controller interconnect driver)
  • sdpib driver (B) (Infiniband Sockets Direct Protocol)

  • todm5823 driver (B) (TOD driver module for ALI M5823 and compatible devices)
  • uata driver (B) (IDE HBA driver)
  • usbser_edge driver (B) (Digi Edgeport USB to serial converter driver)

Other Components

  • kcfd (B) (Crypto code withheld due to US export controls on open cryptographic interfaces included in the Solaris binary product)

  • iconv/kbdcomp/od/pax/patch/printf/sed/tr (B)
  • Kernel lock manager support (B)
  • ksh88 and builtin functions available as separate executables (alias, bg, cd, command, fc, fg, getopts, hash, jobs, kill, read, test, type, ulimit, umask, unalias, wait; XPG4 alias, bg, cd, command, fc, fg, getopts, hash, jobs, kill, read, sh, test, type, ulimit, umask, unalias, wait)
  • labeld command (B)|Enabling method for Trusted Extensions
  • libc_i18n (B)
  • libike (B) (Internet Key Encryption Code)
  • localedef command and library (B) (Locale environment definition support)
  • SNMPD (B) (Simple Network Management Protocol daemon)
  • XPG4 versions of more/od/sed/tail/tr (B)
