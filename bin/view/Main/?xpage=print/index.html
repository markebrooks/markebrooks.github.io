Welcome to the OpenSolaris.org site preview

It is important to point out right at the start that this is very much a work-in-progress, and is nowhere near complete. The intent is to give an early development preview of the new 'content editing' capability that will, in time, be applied to opensolaris.org.

There is no real content on this site - only the site 'structure' has been created to illustrate how the spaces and pages for projects, communities and user groups will work with this new 'content editing' capability. Note that this structure fits with the proposed OpenSolaris role restructuring. Anything posted here will be discarded as the site develops and moves towards completion, when the exisiting opensolaris.org content will be migrated.

The 'content editing' capability is supplied by XWiki, and this installation is, from a user's perspective, just a straightforward deployment of XWiki Enterprise 1.6.1. It hasn't even had look and feel changes applied, deliberately to emphasize the point that this is NOT a 'live' site.

While it might just look like XWiki, there are some fairly major changes behind the scenes, as XWiki is integrated with the OpenSolaris auth application.

This means that the auth app acts as the identity repository for XWiki, and XWiki user rights are granted based on the groups to which a user belongs, where group membership is based on the information in the auth app. This allows for flexible rights management - projects, communities and user groups can decide on the access permissions that make sense on an individual group basis. You can read more about the auth app integration here. We have implemented 'option 1'.

If you are a registered opensolaris.org user you should be able to log in, and if you are a leader or contributor (see the new roles) you should be able create and edit pages in the spaces corresponding to the collectives to which you belong. For example, if you are a leader or contributor of Project JKstat, you should be able to create and edit pages under the project_jkstat space. Spaces are listed in the side menu panel "Spaces", and in the index.

Please submit comments to the website-discuss list, bearing in mind that content (including look and feel) issues are not of concern right now. Functional issues (such as being able to login, edit pages and so on) are of concern.
