Arduino

Arduino on OpenSolaris

Arduino is an open-source electronics prototyping platform based on flexible, easy-to-use hardware and software. It's intended for artists, designers, hobbyists, and anyone interested in creating interactive objects or environments. For more information see the Arduino website.

Currently the Arduino development platform doesn't directly support OpenSolaris, and it requires several dependencies such as avr-gcc, the compiler for the Atmel ATmega CPUs used on the boards.  Ed Plese has produced a set of patches to get the Arduino development environment to build on OpenSolaris.

The device drivers needed to communicate with the board were integrated in build 113 so you'll need to be running that build or later.

A binary tarball containing the Arduino development environment and the required dependencies is available on the download centre, see the README.txt for full details.  At present only a x86 tarball is available.

Also included is a snapshot of the source used to create to binaries, see the README.txt for details.
