 == Get Help With OpenSolaris This page describes the various help and contact resources on OpenSolaris.org and related sites. For commercial support of the OpenSolaris OS, please go to opensolaris.com. ==== Communities that Help Documentation  Website  Advocacy  Installation & Packaging  Desktop  Laptop Learning Materials >> New-To-OpenSolaris  >> Developer Getting Started  >> General Help
 == Contact  Have a question about, or need help with, OpenSolaris? People in the OpenSolaris community can assist you.  === How to contact  The OpenSolaris forums, labeled "Discussions" on the site, manage both web-based posting and viewing, in addition to handling mailing list-like functionality for users that choose to 'watch' a forum. There are many OpenSolaris-related forums, and this number will grow with the community. You will find a comprehensive list on the [Discussions](/os/discussions/) page. If you are new to OpenSolaris, we suggest browsing the opensolaris-discuss forum, which like all forums, includes a search feature to access previous discussions. Contact Forums >> Installation Questions opensolaris-help >> General Questions opensolaris-discuss >> Technical Questions opensolaris-code >> Bug Reporting opensolaris-bugs >> Website Support website-discuss
 == Discussions Subscribe to OpenSolaris forums and mailing lists to hear from the experts, review archives, and post your questions. ==== Chat Join one of the channels at irc.freenode.net to connect with users and developers.  Connections >> User Groups  >> Universities  >> Countries
 == Documentation  The Documentation Group provides detailed information on how to participate in OpenSolaris documentation, access to open sourced documents, and pointers to detailed information about setting up and developing OpenSolaris. The OpenSolaris.org Documentation Page provides pointers to documents written by Communities and Projects on the site.  ==== Blogs Search OpenSolaris community member blogs for examples, tips, and tricks. Information Hubs >> Genunix Wiki  >> BigAdmin  >> Current Documentation  >> Big Doc List
