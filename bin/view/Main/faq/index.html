Below are FAQ documents answering questions about the OpenSolaris project, license, and code base. For help with OpenSolaris terms and acronyms, visit the Glossary.

opensolaris.org Website Transition

The Website Engineering Team has been updating the opensolaris.org website infrastructure recently. Two major transitions have been completed in 2009, and others will continue through 2010. The FAQs below explain the key issues, and more information can be found in the Website Community.

  • opensolaris.org Website Transition FAQ
    General issues about the opensolaris.org website transition. Questions are grouped into three sections: General, Auth, and XWiki.
  • Website Transition: User Actions FAQ
    Things users need to do to prepare for various site transitions related to the Auth application, updates to the SCM console, and the move to XWiki. Questions are grouped by topic.
