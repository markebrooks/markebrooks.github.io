User Groups

User Groups are sponsored by the Advocacy Community Group and help people around the world with all skill levels get involved in the community. To join your local user group, see the detailed list that notes locations, leaders, and mailing lists. Also, subscribe to advocacy-discuss and osug-leaders if you have any questions or want to get involved.

To start a new user group, just send a simple proposal to the Advocacy discussion list. Tell us the name and location of your group, the user names of two or more initial participants, and a short description of the group that includes location, activities planned, related organizations, etc. 

The list of user groups below represents only those groups that have spaces on this website. However, some user groups have their own infrastructure elsewhere, so we point to those groups as well in the leaders table.
