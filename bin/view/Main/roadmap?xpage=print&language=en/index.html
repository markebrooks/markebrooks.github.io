Roadmap 2011

Archive: 2011 | 2010 | 2009 | 2005-2008 | Consolidations | Website Announcements

This page outlines website development and operations work for opensolaris.org. Advance notice for deployments and maintenance is given as per the Website Guidelines policy. Report website problems to website-admin AT opensolaris DOT org.

October 2011

Wednesday October 26: Jive forum application decommission.

Tuesday October 18: Bugzilla downtime for maintenance.

Friday October 7: All opensolaris.org domains down for the weekend for data center updates

September 2011

Thursday September 1: Planned Maintenance Outage for repo, hub and www.opensolaris.org.

August 2011

Thursday August 18: Planned Maintenance on multiple website applications.

Wednesday August 10:  Updated XWiki to v2.7.1.

June 2011

Monday June 27: Deployed a new code review tool at https://cr.opensolaris.org//. Also, decommissioned the old code review tool at http://cr.opensolaris.org (old reviews to remain viewable for 3 months).

Wednesday June 15: Updated Bugzilla at defect.os.org to version 3.6.5.

Monday June 13: Deployed a new application at static.opensolaris.org for community-wide use to upload/store files. See Site User Guide for instructions. 

Monday June 6: Notice that Bugzilla at defect.opensolaris.org will be updated and integrated with auth.opensolaris.org for single sign on. See User Actions and Deadlines for details.
