HPC Developer Kit Preview

Welcome to the Sun HPC Developer Preview, May 2008 Release

The Sun HPC Developer Preview, May 2008 Release provides you
the core platforms, tools, and technologies that will enable you to develop high performance computing applications. This integrated product development environment is certified on the robust Solaris operating system. Combining this environment with a vibrant HPC developers community, Sun support, and training services provides a path to success for creating solutions to your challenges. Use the information and resources in this preview to develop, build, deploy, debug and manage your application binaries as well as the source code.

Downloads

Download the Solaris HPC Developer Preview 05/08.

Technologies

The HPC developer platform consists of various technologies and products. The following example illustrates a typical HPC developer platform based on Sun's technologies and built around the three core products that are part of this distribution:

--

Sun(TM) Grid Engine 6.1

The core of the HPC developers platform is the Sun Grid Engine product, which provides the capability to achieve policy-based workload management and dynamic provisioning of application workloads. When deployed in high intensity computing environments, the Sun Grid Engine software takes the responsibility for accepting, scheduling, dispatching, and managing the remote execution of large numbers of standalone, parallel or interactive user jobs. It also manages and schedules the allocation of distributed resources such as processors, memory, disk space, and software licenses.

 Learn about Sun Grid Engine software:

  • Administration Guide
  • User's Guide
  • Sun Grid Engine Homepage
  • Open Grid Engine Project

--

Sun(TM) HPC ClusterTools 7.1

The HPC ClusterTools product enables users to run complex, compute-intensive MPI applications on Sun systems and clusters of Sun systems. Sun HPC ClusterTools 7.1 software is based on the open-source Open MPI 1.2.4.

 Learn about HPC ClusterTools software:

  • Migration Guide
  • User's Guide
  • Using the Solaris DTrace utility with Open MPI application
  • Sun HPC ClusterTools Homepage

--

Sun(TM) Studio 12

The Sun Studio 12 software provides the application development capabilities for your high-performance application needs. These features include a host of optimizing compilers, an integrated developmental environment (IDE) based on NetBeans 6.0 and plugins that are specific to carrying out interactive functions on Sun Grid Engine clusters, such as deploying and monitoring tasks.

 Learn about Sun Studio software:

  • Sun Studio 12 Documentation
  • Sun Studio Homepage
