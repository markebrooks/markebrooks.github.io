The Sun HPC Software, Editions for OpenSolaris

The motivation for HPC Stack for OpenSolaris project is to simplify traditional software application development and deployment for HPC clusters. It seeks to assemble all the best practices and tools available for HPC practitioners on the OpenSolaris platform into a set of fully integrated offerings.

Illustration 1: This diagram shows the OpenSolaris HPC projects as two deliverables, a developer release (June,2009) and a deployer release (1H-2010).

The goal of the project is to address the entire OpenSolaris HPC cluster software lifecycle, from applications development to the initial build out and maintenance of the cluster itself. By partitioning the effort into two major deliverables, developer and deployer, the project team is able to deliver usable tools faster following agile software development practices. Further, by combining the best of Sun owned software development and deployment tools, alongside commonly accepted and widely available open source HPC software, software developers, system adminstrators and end users benefit directly in the saving of untolled amounts of time installing, configuring and integrating dozens of individual software packages.

Illustration 2: This diagram depicts the stack as a collection of existing Sun owned and open source HPC software with the common thread of OpenSolaris as the base deployment platform.

Sun HPC Software, Developer Edition 1.0 for OpenSolaris

Sun HPC Software, Developer Edition 1.0 for OpenSolaris is a full-featured HPC development environment distributed as a virtual machine (VM). The VM is pre-configured as a grid enabled, virtual HPC cluster comprised of three OpenSolaris zones. This software gives OpenSolaris developers a turn-key parallel application development environment with distributed resource management and cloud connectivity built in and ready to go:

At a Glance

  • OpenSolaris and common open source HPC software.
  • Sun Studio C, C++ and Fortran compilers and tools.
  • Sun HPC ClusterTools.
  • Sun Grid Engine.
  • Sun Open Storage solutions.
  • Sample applications are included in the installation to get you up and running quickly.

OpenSolaris Deployer

Work has begun on a complete HPC cluster deployment system running OpenSolaris on the head node(s) and compute nodes. This offering will allow HPC cluster administrators to deploy and manage large scale HPC clusters running OpenSolaris. This project is expected to deliver in the first half of calendar year 2010 (1H-2010).
