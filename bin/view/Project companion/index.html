Contributor information and guidelines

 The Companion project is now live with a Subversion repository as well as a buildable source tarball and binary and source packages created from that tarball.

 The Companion project is a home for externally derived open source software intended to complement OpenSolaris distributions. Several different groups pursue this goal today, including Blastwave, SunFreeware, and the Companion.

 The Companion project aims to explore requirements and possibilities for cooperation among the various existing projects as well as engage new community members.

 Access to the subversion repository, which is the Companion Software build environment, is via svn+ssh. Anyone may checkout the repository as user "anon" as shown below.

 % svn co svn+ssh://anon@svn.opensolaris.org/svn/companion/core/usr

 For more information about using subversion see  SVN Help. 

 If you would like to contribute software into the Companion, you will need to affiliate with the project first, then please review the  Contributor Guidelines page.

 You can subscribe to the  companion-discuss@opensolaris.org mailing list or join the discussion through the companion-discuss web forum.

 To contribute to the Companion, subscribe to the alias  companion-integrate@opensolaris.org.
