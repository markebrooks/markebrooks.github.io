Zones Project Documents

Zones Project Documents.  

Comments and feedback are welcome.  The RFE/Feature label links to the discussion of that topic.

_RFE/Feature_Status_Description
 S10 brand project case  Under development  A brand which enables running Solaris 10 in a zone on OpenSolaris  
 max-processes rctl case  Under development  Enables limits for process table slots 
 Native p2v case discussion  Integrated Feb. 2009 in Nevada build 109  Enables installing a native zone from an image of a physical system (physical to virtual)  
 Configurable Host IDs case  Integrated Jan. 2009 in Nevada build 108  Enables per-zone hostid different from the global zone 
 zoneadm attach -b option case  Integrated July 2008 in Nevada build 96  Enables patch backout during zone migration 
 solaris 9 brand (closed case)  unbundled  A new solaris9 brand 
 update on attach case  Integrated Jan. 2008 in Nevada build 82  Enables update during zone migration 
 solaris 8 brand (closed case)  unbundled  A new solaris8 brand 
 rcapadm zone option case  Integrated Dec. 2006 in Nevada build 56  Modifies 2006/496 
 swap rctl & locked mem RM case  Integrated Dec. 2006 in Nevada build 56  This project adds a swap rctl to Zones. 
 improved zones/RM integration case, discussion  Integrated Dec. 2006 in Nevada build 56  This project improves the integration of Zones and resource management. 
 Zone boot arguments II case, discussion  Integrated June 2006 in Nevada build 44  Zone boot argument passing 
 Zones migration dry-run case, discussion  Integrated May 2006 in Nevada build 42  This case address an RFE to provide a dry-run for zone migration. 
 Zones ZFS support case, discussion  Integrated April 2006 in Nevada build 39  This case address an existing RFE to provide integrated support for using ZFS snapshots and clones when cloning zones. Additionally, this case enhances zones to automatically create ZFS filesystems during zone installation and destroy ZFS filesystems during zone uninstall. 
DTrace support (4970596)  Integrated March 2006 in Nevada build 37  This RFE, in conjuction with 4966416  and 6231905 allows the use of certain DTrace providers within a non global zone.  
Configurable privileges for zones  Integrated March 2006 in Nevada build 37  This case proposes a new facility for zones: the ability to specify the set of privileges that all of the processes in a zone are limited to. 
 Zones migration (attach/detach) case, discussion  Integrated March 2006 in Nevada build 36    This change provides the ability to migrate an installed non-global zone from one machine to another. 
 logadm $zonename keyword case, discussion  Integrated March 2006 in Nevada build 36  (Contributed by Rich Lowe) This document proposes the addition of a new keyword for the logadm[1] utility, $zonename. 
 Zones move and clone case, discussion  Integrated Feb 2006 in Nevada build 33  This fast-track enhances the Solaris Zones subsystem to address two existing  RFEs. The first enables non-global zones to be relocated from one point on the filesystem to another; this may involve actually moving the bits and requires changing the associated metadata (the "zonepath"). The second enables administrators to rapidly provision new non-global zones, once one has been set up, by allowing installation via copying from another (non-global) zone. 
 Zones Rename: Case, Discussion  Integrated Sept 2005 in Nevada build 24  This case proposes a new facility for zones: rename of non-global zones. 
 Raise Commitment Level for zone_enter: Case  N/A  This case raises the commitment level of zone_enter(2) to "Contracted Project Private" 
 Single User Zone Boot Case  Solaris 10 build 66  Adds zoneadm -z <zonename> boot -s 
 /dev/openprom in a Zone Case  Solaris 10 build 60  Adds /dev/openprom to zones on SPARC 
 Zone extensions for ps(1) Case  Solaris 10 build 60  Adds ps -z and -Z 
 Zones Initial Case Case  Solaris 10 build 50  The main zones case
