Getting Started with IPS for 2008.11

Image Packaging System Documentation

 The Image Packaging System (IPS) software is developed by the Indiana and IPS OpenSolaris project teams and is a part of the OpenSolarisTM 2008.11 release.
Click on the link below to download and install the OpenSolaris 2008.11 release.
Get the OpenSolaris 2008.11 Release
Read the IPS Guide  to learn about the latest features of IPS for 2008.11.
 Read the Setting up Your Development Environment on OpenSolaris 2008.11 System  to get started on developing applications with AMP, Java, Ruby, C, and C++ .

Developer Topics

If you are a developer with some knowledge of packaging in general and IPS in particular, and would like to skip the basics, the following topics might be of interest to you.

  • How to create and Publish an IPS Package
  • How to Create a Repository
  • How to Publish a SVR4 Package to the Repository
  • How to Get the Developer Tools
  • Using Sun Studio with OpenSolaris
    SVR4 Packaging
    The OpenSolaris 2008.05 release supports both SVR4 packages and IPS packages. To understand the SVR4 packaging model,read the Application Packaging Developer's Guide for detailed information on creating SVR4 packages.
     Read the Comparison Table  to view the differences between IPS commands and apt-get packaging commands.
    Background Reading
    The project team has been explaining some of the ongoing assumptions behind the project in a series of blog posts:
    • Bart on Rethinking patching, and
    • Stephen on Observations on packaging, pkg(5): a no scripting zone, pkg(5): leaving the build system `out', and pkg(5): no more installer magic.
