 January 15, 2008  Project Indiana  Bob Netherton  Our guest speaker for this meeting will be Bob Netherton, who will be talking about Project Indiana. The idea behind Project Indiana is to deliver an OpenSolaris binary distribution in a more modern fashion than that of traditional Solaris. Rather than requiring the download of the entire body of code, the installer in its current form is a single "live CD" image and installer. The new installer is attractive and easy to use. Once installed, you'll be running from a native ZFS file system. Project Indiana also will adopt the use of network repositories and related package management system--additional installation of packages can come from the network. And it's all publicly redistributable and mirrorable.  -

 November 20, 2007  RBAC  Dale Somer  The Solaris Role-Based Access Control (RBAC) facility provides a robust way to limit and control user access to commands and functions in the Solaris Operating Environment.  RBAC concepts and definitions will be discussed along with a view of the four primary configuration files used in the facility.  The presentation will include a demonstration of using "roles", "profiles" and "authorizations" to provide access to administrative commands like "ifconfig" and "svcadm".  In addition, the Solaris 10 Process Rights Management (privileges) will be discussed and examples of using privileges to enable capabilities like dtrace for non-root users will be shown.  Finally, RBAC and sudo will be compared and examples of how RBAC can help the Solaris system administrator delegate tasks and have more time to focus on other complex administrative projects.  "RBAC Demystified" slides

 April 17, 2007  Live Upgrade  Jeff Logullo, Steve Smith  Live Upgrade allows system administrators to perform both system patching and system upgrades without incurring system downtime. Live Upgrade is the safest and least obtrusive method to perform upgrades. And, if an upgrade does not go smoothly, it is a simple matter to return the server to its previous state. Live Upgrade is a boon for servers with high uptime requirements. It also is a big help for Open Solaris users that want to stay on the cutting edge, upgrading to the latest Open Solaris Community Edition monthly builds without losing any customization.  -
