മലയാളം ഗ്രൂപ്പ്‌

ആമുഖം

 Open Solaris എന്ന ലോകനിലവാരത്തിലുള്ള Operating System ത്തെ മലയാളത്തിന്റെ അനന്തസാധ്യതകളേയ്ക്കു കൈപിടിച്ചുയര്‍ത്താനുള്ള ഒരു കൂട്ടായ്മയാണിത്‌.

ലക്ഷ്യങ്ങള്‍

  • ഭാഷയ്ക്കനുസരിച്ച്‌ യന്ത്രത്തേയും തന്ത്രത്തേയും രൂപപ്പെടുത്താനുള്ള ഭാഷാസ്നേഹികളുടെ പോരാട്ടങ്ങളില്‍ പങ്കാളിയാവുക. വികലമായ അളവുകോലുകള്‍ക്കനുസരിച്ചു കീറിമുറിയ്ക്കപ്പേട്ട ഭാഷയെ പുനരുജ്ജീവിപ്പിക്കുക.
  • മലയാളികള്‍ക്ക്‌ അര്‍ത്ഥസമ്പൂര്‍ണ്ണവും പരിപൂര്‍ണ്ണവുമായ മലയാള desktop അനുഭവവേദ്യമാക്കുക.
  • ഇതിനായി തുറന്നതും സ്വതന്ത്രവുമായ മാനദണ്ഡങ്ങളേയും ഏകകങ്ങളേയും അടിസ്ഥാനപ്പെടുത്തി OpenSolaris software കള്‍ നിര്‍മ്മിക്കുക.
  • Open Solaris ലെ പ്രധാനപ്പെട്ട എല്ലാ applications ഉം ഗുണനിലവാരം ഉറപ്പാക്കിയ അനുയോജ്യമായ മലയാള തര്‍ജ്ജമകള്‍ നല്‍കുക.
  • Open Solaris ഉപയോഗിക്കാന്‍ താല്‍പ്പര്യമുള്ള ഭാഷസ്നേഹികളെ സഹായിക്കുക.
     മലയാളത്തേയും സ്വതന്ത്ര computer സംസ്ക്കാരത്തേയും സ്നേഹിക്കുന്ന എല്ലാവരേയും ഈ കൂട്ടായ്മയിലേയ്ക്കു സ്വാഗതം ചെയ്യുന്നു.

ഈമെയില്‍ വിലാസങ്ങള്‍

  • g11n-ml-discuss കൂട്ടായ്മയിലെ എല്ലാ അംഗങ്ങളും ഉള്‍പ്പെടുന്ന ഈമെയില്‍ ഗ്രൂപ്പ്‌

നിലവില്‍ പരിഗണയിലുള്ള സംരംഭങ്ങള്‍

 projectന്റെ പേര്‍  വിവരണം  മുഖ്യ സന്നത്ഥസേവകന്‍  കൂട്ടായ്മയുടെ ഈമെയില്‍ വിലാസം
desktop applications മലയാള തര്‍ജ്ജമകള്‍മലയാള തര്‍ജ്ജമകളും അവയുടെ നിലവാര പരിശോധനയും.സുരേഷ്‌ ചന്ദ്രശേഖരന്‍g11n-ml-translations

Bugs and RFEs

 മലയാളം Open Solaris ഉപയോഗിക്കുമ്പോള്‍ കാണുന്ന പ്രശ്നങ്ങളും പിന്നെ ഭാവിയില്‍ നിങ്ങള്‍ ആഗ്രഹിക്കുന്ന നൂതന സംവിധാനങ്ങളേയും കുറിച്ച്‌ താഴേ കൊടുത്തിരിക്കുന്ന category/sub-categoryയില്‍ വിവരണങ്ങള്‍ സമര്‍പ്പിക്കുക.
 ഉപയോഗിക്കേണ്ട category/sub-category

  • library l10n-common
  • for any translated messages at system libraries, Malayalam locale system libraries, and the locale itself.
  • utility l10n-common
     for any translated messages at commands and utilities, and Malayalam applications.
     സാധ്യമെങ്കില്‍ bugs report ചെയ്യുന്നതിനു മുന്‍പായി അതേ തരത്തിലുള്ള bugs നിലവില്‍ ഇല്ല എന്നുറപ്പുവരുത്തുക.
     നന്ദി
