OpenSolaris Binary Distro

OpenSolaris Binary Distro is working towards creating a binary distribution of an operating system built out of the OpenSolaris source code. The distribution is a point of integration for several current projects on OpenSolaris.org, including those to make the installation experience easier, to modernize the look and feel of OpenSolaris on the desktop, and to introduce a network-based package management system into OpenSolaris.

The resulting distribution is a live-CD install image, and is fully permissible to be redistributed by anyone. It will also have the capability for developers to create their own, customized distribution based on OpenSolaris Binary Distro.

Documentation at openSolaris.org

 
Main G11n OpenSolaris binary distro test goal is to test entire distribution and ensure the system is working and is usable from global perspective, taking into account wide variations in regions, languages, and cultures. Localization (l10n) and internationalization (i18n) of OpenSolaris operating system introduces multilingual support to the product through the addition of locale support and offers language specific support for the GUI for pre-selected languages and regions. This can result in certain issues and non-expected behavior that needs to be tested

How to Report New Bug & Bug Queries

To report bug, use OpenSolaris Bugzilla tracking system. If you don't have already an account, you can look at Create a Bugzilla Account and very easily open a new one. When submitting a new issue, please follow Bug Writing Guidelines and for any unclearness you can consult The Bugzilla Guide - Filing Bugs or contact G11n Test Group g11n-test-group@opensolaris.org. To submit G11n issue use i18n-l10n component and also put appropriate Whiteboard keyword (see below section). Please search in bugzilla first to avoid submitting of already known problem (you can use below queries to see current issues).

  • Bugzilla
  • Bugzilla - Known Issues
  • Issues tracked in Bugster

Whiteboard Keywords:

  • app-i18n - For i18n issues for applications with gui
  • x-i18n - For X i18n issues, excluding keyboard issues
  • font-i18n - For font i18n issues
  • input-i18n-xkb, input-i18n-iiim, input-i18n-ibus, input-i18n-scim - For input i18n issues - including xserver/xkb and all input method frameworks
  • cli-i18n - For command line utilities i18n issues
  • lib-i18n - For system library i18n issues
  • installer-i18n - For i18n issues connected with software installation or upgrade and tools around that, mainly installer
  • locale-i18n - For locale data and locale functionality issues
  • software-l10n - For all l10n issues (wrong or missing translation)
  • doc-l10n - For l10n issues in document translations

Part of G11n Test Group effort is also to verify fixed bugs and to provide more information if requested by assignee:

  • bugzilla - g11n INCOMPLETE bugs
  • bugzilla - g11n bugs to be verified and closed
