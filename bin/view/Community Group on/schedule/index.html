ON notes

  1. Gate closes at 11:00 PM Pacific on the date specified in that column.
  2. Projects need to be delivered before COB on the Friday before the build.
  3. * notes holidays that delay the build to Tuesday.
  4.  notes a three week build due to US Holiday shutdowns

General notes

  • The SXCE binary releases are built from the above Nevada builds and are typically released about 10 days after the WOS build closes.
  • Manpage and Docs CR's need to be in the committed state before putback
  • No release date has been set yet, so:
