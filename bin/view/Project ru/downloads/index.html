Использование OpenSolaris

Если вы хотите загрузить и запускать на своем компьютере дистрибутив, основанный на OpenSolaris, выберите один из следующих вариантов:
Solaris Express Community Edition – DVD: загрузить | CD: загрузить

 Solaris Express Community Edition - это двоичный релиз компании Sun, предназначенный для разработчиков (имеющий кодовое название "Nevada"). Он содержит самые свежие, неподдерживаемые двоичные модули OpenSolaris, а также дополнительные технологические компоненты, которые еще не выпущены в исходных текстах в составе OpenSolaris. Разработчики могут компилировать исходные тексты OpenSolaris, используя этот релиз в качестве базовой системы. Он обновляется каждую вторую пятницу.

Solaris Express Developer Edition - Загрузить | Дополнительная информация

Solaris Express Developer Edition - это официально выпускаемый Sun релиз, включающий двоичные модули OpenSolaris и дополнительные технологические компоненты, которые еще не были выпущены в составе исходных текстов OpenSolaris. Компания Sun предлагает ограниченную поддержку этого релиза. Он предназначен преимущественно для клиентов Sun с целью тестирования последних технологических решений, которые в конечном итоге будут выпущены в составе продукта Sun. Solaris Express обновляется один раз в месяц и доступен для свободной загрузки.

BeleniX – Загрузить | Дополнительная информация

 BeleniX - это дистрибутив *NIX, который строится с использованием исходных текстов OpenSolaris. В настоящее время это дистрибутив формата LiveCD, но планируется, что в будущем он перерастет в полный дистрибутив, допускающий установку на жесткий диск.

MartUX – Загрузить | Дополнительная информация

 marTux - это первый (после Solaris Express/Solaris Express Community Release) дистрибутив OpenSolaris для архитектуры SPARC (для sun4u в настоящее время, версия для sun4v появится позже).

NexentaOS – Загрузить | Дополнительная информация

 NexentaOS - это полная основанная на проекте GNU свободная операционная система с открытым кодом, построенная поверх ядра и системных утилит OpenSolaris. NexentaOS интегрирует OpenSolaris (ядро SunOS) и приложения с открытым исходным кодом, соединяя лучшее двух миров.

SchilliX – Загрузить | Дополнительная информация

 SchilliX - это основанный на OpenSolaris Live CD и дистрибутив для архитектур x86, x64 и EM64T. Текущий релиз основан на Nevada Build 17 или более позднем.

Изучение OpenSolaris

В рамках проекта OpenSolaris был подготовлен стартовый комплект для разработчиков, которые хотят научиться устанавливать, загружать и работать с OpenSolaris. Стартовый комплект включает в себя сравнение OpenSolaris с другими операционными системами.

Сборка OpenSolaris

Проект OpenSolaris состоит из множества различных программных компонентов, крупнейшие из которых называются "консолидациями". Консолидация "Operating System/Networking" (ON) является базовой системой для OpenSolaris. Для того чтобы собрать OpenSolaris из исходных текстов, вам нужно сначала установить подходящий дистрибутив OpenSolaris, список которых в настоящее время ограничен только Solaris Express Community Release (см. выше).

Консолидации OpenSolaris

Множество разных программных компонент составляют проект OpenSolaris. Крупнейшие компоненты называются "консолидации". Консолидация "Operating System/Networking" (ON) является базовой системой для OpenSolaris.

Operating System/Networking (ON)

 ON сключает в себя ядро, сетевые службы, а также системные библиотеки и команды.

Дополнительные консолидации OpenSolaris:

Application Server

 Сервер приложений Sun Java System Application Server.

Developer Product Tools (DevPro)

 В состав консолидации DevPro входят такие компоненты: системная математическая библиотека, библиотека для работы мультимедиа-форматами, библиотека microtasking, SCCS, make и библиотка времени выполнения для программ на C++. Некоторые из них доступны в виде исходных текстов, а другие - только в двоичном виде.

Documentation (Docs)

 Техническая документация для разработчиков и администраторов. Доступны исходные тексты многих книг. Дополнительную информацию можно найти на странице загрузки.

Globalization Support (G11N)

 Поддержка интернационализации и локализации.

Installation Support (Install)

 Поддержка инсталляции и инструменты управления пакетами.

Java Desktop (JDS)

 Безопасное и всеобъемлющее программное решение для корпоративной настольной системы.

Java Platform, Standard Edition (Java SE)

 Включает двоичные модули Java Development Kit (JDK) и Java Runtime Environment (JRE).

Manual Pages

 Исходные тексты справочных руководств SunOS.

Message Queue

 Система гарантированной доставки сообщений Sun Java System Message Queue. Доступен архив формата zip с исходными текстами.

Network Storage (NWS)

 Поддержка сетевых устройств и систем хранения данных.

SFW (Solaris FreeWare)

 Программное обеспечение с открытым кодом, которое поставляется совместно с Solaris/OpenSolaris.

SPARC Graphics Support

 Консолидация "SPARC graphics consolidation" (Поддержка графики на платформе SPARC) содержит драйверы, доступные только в двоичной форме.

Test

 Наборы тестов и инструменты тестирования OpenSolaris.

X Window System (X11)

 Программное обеспечение системы X11.

В будущем будут доступны дополнительные консолидации OpenSolaris. Информацию об этом можно найти на странице Roadmap.

Инструменты

Для сборки исходных текстов OpenSolaris рекомендуется использовать компиляторы Sun Studio 11 . Некоторые консолидации работают над возможностью использования компилятора GCC в дополнение к Sun Studio. Детали можно найти на конкретных страницах загрузки.
