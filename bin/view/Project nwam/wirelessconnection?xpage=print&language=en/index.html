Wireless Network Connection

 While the wireless network connection's prototypical location is as an object in the Network Connections folder, an icon representing it and its current state can be shown in the system tray as well.

Wireless Network Connection System Tray Icon

 When the computer is first opened and the wireless networking turned on via a switch on the laptop, the following information balloon appears.

 This is a tooltip you will get if you hover over the wireless icon in the system tray.

 The contextual menu for the same System Try icon.

 When a connection is made, you may get this baloon:

 A message indicating no wireless.

Available Wireless Networks Window

 Clicking on the baloon declaring that there are available wireless networks or choosing the "View Available Wireless Networks" menu item will display this window. This displays all the local wireless networks along with their security status and their strength. Their order seems to be a "first come, first displayed" scheme (each click on refresh will show them in a different order).

 Note that computer-to-computer wireless networks get a different icon:

 Clicking on a network expands it a bit to show more information.

 Clicking on Refresh clears the list and after several seconds redisplays.

 Clicking on Set up a wireless network for a home or small office displays the Wireless Network Setup Wizard.

 Clicking on "Learn about wireless networking" displays some online help. "Change the order of preferred networks" displays the Wireless Networks tab of the Wireless Network Connection Properties window, while clicking on "Change advanced settings" displays the General tab of the same window.

 If, instead, one clicks on an insecure network and then on the "Connect" button, one is shown this message box:

 Affirming that message box will give you one of two bits of feedback. If you have no active connection, then the icon in the System Tray animates:

 Alternately, if you already have a connection, then you will be shown a message box like this with animation between the two computers.

 As this goes on, the message in the main window may change to notify of changes of state:

 On successful connection, the message in the window will change:

 And the system tray icon will show connection (the screen in the icon seems to blink periodically when there is traffic on the net).

 if you now try to disconnect, you will get this message:

 Connecting to another address (rather than disconnecting) leaves an indication that both are special in some manner.
