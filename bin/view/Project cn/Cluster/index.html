Open High Availability Cluster (OHAC) 是 Sun Microsystems, Inc. 开发的高可用性 (high availability, HA) 群集解决方案 Solaris Cluster1 的开放源代码库。Solaris Cluster 包括核心群集框架、一套针对各种应用程序的 HA 代理、自动测试框架以及灾难恢复扩展功能。
虽然刚开始只是发布一部分代码，但以后会陆续通过 Open HA Cluster 发布其他部分。此开源计划将会是在社区众人参与的情况下开展，并且有可能为以后的 Solaris Cluster 产品组版本提供发展基础。

与 Solaris Cluster 有何不同？
 Open HA Cluster 与 Solaris Cluster 的主要区别在于 Open HA Cluster 并不提供最终用户产品，也不提供完整的分发产品。它其实是一个开放源代码库，并包含开发、编译和使用代码时必需的一些构建工具。它提供了一个用于交流和共 享相关信息的基础结构，以及一个可以通过社区协作方式向代码库中添加最新先进功能的平台。
社区将会提供代码支持，但 Sun 并不为 Open HA Cluster 产品提供正式支持，无论是源代码形式还是二进制形式。
Solaris Cluster 产品组是 Sun 的品牌产品，由 Sun 测试、维护以及支持。可以利用 Open HA Cluster 源代码构建以后的 Solaris Cluster 版本，但将会与当前版本的 Solaris Cluster 产品一样为其提供全面支持。然而，主要目的是通过 Open HA Cluster 发布尽可能多的现有源代码，并且由 HA Clusters 社区推动未来的开发工作。Open HA Cluster 和 Solaris Cluster 产品的内容在任何时候都会有所不同。

  • 首先浏览 Open HA Cluster 源代码。
  • 下 载和编 译源代码。
  • 如果您想深入了解更多信息，请访问 Open HA Cluster 文 档页面，获取详细的文档、幻灯片、TOI 以及其他资源。
  • 在 HA Cluster：讨论的 讨论组以及各种其他 OpenSolaris 讨 论组中结识其他相关人士并进行交流。
  • 如果您发现任何问题，请单击此处报告问题。
  • 如果您想为社区做出贡献，请遵循贡献者指南中的说明或者直接 跳至潜 在项目列表。
  • 如果您遇到困难或有不清楚的地方，请参阅常 见问题解答。如果仍然无法解决，请搜索 HA Cluster：讨论归档或 发送邮件至 ha-clusters-discuss@opensolaris.org。
