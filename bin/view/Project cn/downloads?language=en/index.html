使用 OpenSolaris

如果您需要运行基于 OpenSolaris 的发行版本，请从下列选项选择：
Solaris Express, Community Release – DVD: 下载 | CD: 下载

 Solaris Express, Community版是Sun为开发者提供的一个发行版本(代码名称“Nevada”)。其包含了最新的、无技术支持的 OpenSolaris 产品，以及尚未在 OpenSolaris 源代码库中发布的其它技术。开发者可以通过使用该版本生成OpenSolaris源代码作为基础系统。该版本每周五更新。

Solaris Express - 下载 | 更多信息

Solaris Express是Sun OpenSolaris的官方发行版本，包含尚未在OpenSolaris源代码库中发布的其它技术。Sun对该版本提供一定的技术支持。该版本主要用于Solaris客户尝试最新技术，这些技术最终将由Sun产品化。Solaris Express每月更新并提供免费下载。

BeleniX – 下载 | 更多信息

 BeleniX是一个使用OpenSolaris源代码库生成的*NIX 发行版本。目前提供LiveCD分发，也即将提供完整的可安装至硬盘的发行版本。

marTux – 下载 | 更多信息

 marTux是为SPARC(目前为sun4u, 以后将是sun4v) 提供的第一个非Solaris Express/Solaris Express Community 版的OpenSolaris 发行版本。

NexentaOS – 下载 | 更多信息

 NexentaOS是一个在OpenSolaris内核和运行时之上生成的操作系统，完全基于GNU 提供免费和开放源代码。NexentaOS集成了OpenSolaris (SunOS内核) 和Open Source Software (OSS)应用程序 ，包综合了两部分的优点。

SchilliX – 下载 | 更多信息

 SchilliX是为x86, x64和EM64T架构提供的基于OpenSolaris的UNIX Live CD及发行版本。当前发布基于Nevada Build 17及以上。

学习 OpenSolaris

 OpenSolaris项目已经为希望学习如何安装、启动和运行OpenSolaris的开发者制作了初学者工具。初学者工具包括OpenSolaris和其它操作系统的比较。

生成 OpenSolaris

 OpenSolaris项目包括许多不同的软件和组件，主要的组件称作“consolidations (部件)”。Operating System/Networking (ON)部件提供OpenSolaris的基础系统支持。要从源代码生成OpenSolaris，您首先需要安装配套的OpenSolaris 发行版本，目前为Solaris Express Community 版 (如上)。

OpenSolaris部件

 OpenSolaris项目包括许多不同的软件和组件，主要的组件称作“consolidations (部件)”。Operating System/Networking (ON) 部件提供OpenSolaris 的基础系统支持。

Operating System/Networking (ON)

 ON 包括内核、网络服务和系统程序库与指令集。

其它 OpenSolaris 部件：

Application Server (应用程序服务器)

  Sun Java System Application Server (Sun Java 系统应用程序服务器)。

Developer Product Tools (DevPro) (开发者产品工具集)

  DevPro 部件含有多个组件：系统数学库、媒体库、微任务库、SCCS 与make，以及C++运行库。其中一些提供开放源代码，还有一些提供二进制文件。

Documentation (Docs) (技术文档)

  开发者和系统管理员技术文档。提供多本书籍源代码的下载。详情请参见下载页。

Globalization Support (G11N) (全球化支持)

  国际化和本地化支持。

Installation Support (Install) (安装支持)

  安装支持和打包工具。

Java Desktop (JDS) (Java桌面)

  安全、综合的企业桌面软件解决方案。

Java Platform, Standard Edition (Java SE) (Java平台标准版)

  提供 Java Development Kit (JDK) 二进制文件和 Java Runtime Environment (JRE) 下载。.

Manual Pages (用户指南)

  SunOS Reference Manual Pages (SunOS 用户参考指南)源代码。

Message Queue (消息队列)

  Sun Java System Message Queue (Sun Java系统消息队列)。提供开放源代码 zip 文件下载。

Network Storage (NWS) (网络存储)

  网络存储设备支持。

SFW (Solaris FreeWare) (Solaris自由软件)

  和Solaris/OpenSolaris捆绑的开放源代码软件。

SPARC Graphics Support (SPARC图形支持)

  提供 SPARC 图形部件二进制形式驱动下载。

Test (测试环境)

 OpenSolaris Test Suites (OpenSolaris测试套件) 和 Test Tools (测试工具集)。

X Window System (X11) (X Window系统)

  X11软件。

 更多OpenSolaris部件陆续提供。具体信息参见Roadmap 。

工具

 推荐使用Sun Studio 11 编译器生成OpenSolaris 。一些部件用于支持GCC，作为 Sun Studio 的补充。指定下载页将提供具体信息。.
