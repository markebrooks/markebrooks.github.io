Installation Instructions

Use the standard bfu procedures  to install these bits on top of an existing Solaris Nevada installation.  We suggest the most recent release of the Solaris Express Community Release.

See the download links above for a link to the ONbld package.

Step by step, here's how to install the Project Clearview bfu bits over an existing Solaris installation:

Example:
The files cv-example-build.sparc.tar.bz2 and cv-example-SUNWonbld-build.sparc.tar.bz2

were downloaded to /tmp/cv_tmp.

/usr/bin/bunzip2 cv-example-build.sparc.tar.bz2  
/usr/bin/bunzip2 cv-example-SUNWonbld-build.sparc.tar.bz2  
/usr/bin/tar xvf cv-example-build.sparc.tar  
/usr/bin/tar xvf cv-example-SUNWonbld-build.sparc.tar.bz2  
/usr/sbin/pkgadd -d /tmp/onbld SUNWonbld 

/opt/onbld/bin/bfu /tmp/cv_tmp/cv-example-build.sparc

One can use acr script to resolve conflicts (described on the bfu procedures  page):

bfu# acr

Check the log file listed in the acr output. If conflicts still
exist edit them manually. If not then reboot.

bfu# reboot
