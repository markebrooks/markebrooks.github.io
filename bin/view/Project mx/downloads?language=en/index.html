Centro de Descargas OpenSolaris

La mayoría del código de OpenSolaris es liberado bajo una licencia Common Development and Distribution License (CDDL) Versión 1.0.
 Código fuente basado en proyectos de código abierto seguirá estando disponible con sus licencias actuales. Algunos componentes binarios están cubiertos por la licencia OpenSolaris Binary License y otros están cubiertos por otras licencias de código abierto. Las páginas de descargas proporcionan información específica de las licencias asociadas con los componentes disponibles.

Usando OpenSolaris

Si quieres ejecutar una distribución basada en OpenSolaris, elige una de las siguientes opciones disponibles:
Solaris Express Community Edition – DVD: Descargar | CD: Descargar

 Solaris Express Community Edition es una versión binaria de Sun liberada para desarrolladores (código de proyecto "Nevada"). Esta contiene la última versión no soportada de OpenSolaris así como tecnología adicional que no ha sido liberada en el código fuente de OpenSolaris. Los desarrolladores pueden construir código fuente de OpenSolaris usando esta versión como base. Esta es actualizada cada Viernes.

Solaris Express Developer Edition - Descargar | Más Información

Solaris Express Developer Edition es una versión oficial de Sun de OpenSolaris con tecnología adicional que no ha sido incluida en el código de OpenSolaris. Sun ofrece soporte limitado para esta versión. El objetivo principal de esta versión es que los clientes de Solaris prueben la más reciente tecnología que eventualmente será liberada por Sun en su producto. Solaris Express es actualizado mensualmente y los binarios están disponibles para descarga gratuita

BeleniX – Descargar | Más Informacion

 BeleniX es una distribución de *NIX que es contruida usando el código fuente de OpenSolaris. Esta es actualmente una distribución LiveCD pero se pretende que crezca a una distribución completa que pueda ser instalada en un disco duro.

marTux – Descargar | Más Información

 marTux es la primera distribución de OpenSolaris (non-Solaris Express/Solaris Express Community) para SPARC (por ahora para sun4u, más adelante para sun4v).

NexentaOS – Descargar | Más Información

 NexentaOS es un sistema operativo completo gratis y de código abierto basado en GNU construido sobre el kernel y ejecutable de OpenSolaris. NexentaOS integra OpenSolaris (kernel de SunOS) y aplicaciones Open Source Software (OSS), esta integración combina lo mejor de los dos mundos.

SchilliX – Descargar | Más Información

 SchilliX es un UNIX basado en OpenSolaris en distribución Live CD para arquitecturas x86, x64 y EM64T. La versión actual está basada en el Build 17 y superiores de Nevada.

Aprendiendo OpenSolaris

El proyecto OpenSolaris ha creado un kit de inicio para desarrolladores que quieren aprender como instalar, levantar y ejecutar OpenSolaris. El kit de inicio incluye una comparación de OpenSolaris con otros sistemas operativos.

Construyendo OpenSolaris

El proyecto OpenSolaris comprende muchos diferentes componentes de software, la mayoria de los componentes son referenciados como "consolidaciones". La consolidación Sistema Operativo/Red (ON) constituye la base de soporte de OpenSolaris. Para construir OpenSolaris desde el código fuente, primero necesitas instalar una distribución adecuada de OpenSolaris, la cual actualmente puede ser solamente la versión Solaris Express Community (ver arriba).

Consolidaciones OpenSolaris

El proyecto OpenSolaris comprende muchos diferentes componentes de software, la mayoria de los componentes son referenciados como "consolidaciones". La consolidación Sistema Operativo/Red (ON) constituye la base de soporte de OpenSolaris.

Sistema Operativo/Red (ON)

 ON incluye, el kernel, los servicios de red y comandos y librerías del sistema.

Consolidaciones Adicionales de OpenSolaris:

Servidor de Aplicaciones

 Sun Java System Application Server.

Herramientas de Desarrollo (DevPro)

 La consolidación DevPro tiene varios componentes disponibles: la librería del sistema matemático, la librería de dispositivos, la librería de microtareas, librerías SCCS y make y librerías de C++ ejecutable. Algunas están disponibles como código fuente y otras como binarios.

Documentación (Docs)

 Documentación técnica para desarrolladores y administradores. El fuente de varios libros está disponible. Ver la página de descargas para más detalles.

Soporte de Globalización (G11N)

 Soporte para Internacionalización y adaptación local.

Soporte para Instalación(Install)

 Soporte para instalación y herramientas de empaquetamiento.

Java Desktop (JDS)

 Una solución empresarial segura y completa de software para el escritorio.

Java Platform, Standard Edition (Java SE)

 Están disponible los binarios para el Java Development Kit (JDK) y Java Runtime Environment (JRE).

Manuales

 El código fuente de los manuales de referencia del SunOS.

Message Queue

 El Sun Java System Message Queue. un archivo zip con el código fuente está disponible.

Almacenamiento de Red (NWS)

 Soporte a dispositivos de almacenamiento de red.

SFW (Solaris FreeWare)

 Software de código abierto que esta empaquetado con Solaris/OpenSolaris.

Soporte de Gráficos en SPARC

 La consolidación de Gráficos para SPARC tiene drivers disponibles en forma binaria.

Pruebas

 Cunjunto de pruebas para OpenSolaris y herramientas adicionales de pruebas.

Sistema X Windows (X11)

 Software de X11.

Consolidaciones adicionales estarán disponibles en un futuro. Ver la Información del Roadmap.

Herramientas

Es recomendado que los compiladores de Sun Studio 11sean usados para construir el fuente de OpenSolaris. Algunas consolidaciones están trabajando para soportar el uso de GCC además de Sun Studio. Las páginas específicas de descargas proporcionan más detalle.
