Sun Tech Days 2008 in Tokyo 終了

12/2 - 12/4 ３日間に渡って東京ミッドタウンにて開催しました、Sun Tech Days 2008 in Tokyo が終了しました。

参加して頂いた方々、およびイベントにご協力して頂いた多くの方々に大変感謝しております。

各セッションの資料はイベントページより、後日ダウンロード出来るようにしますので、今しばらくお待ち下さい。

本日やっと解禁になった JavaFX SDK 1.0 の正式発表や会場で配布していた CD (例のアレ)も、イベントの中では歯にものがつまったような事しか言えなかったりでしたが、参加して頂いた人の特典として用意したノベルティグッズ以上の価値を感じて頂ければ、主催者としては非常にありがたいと思います。

今回は企画だけでなく、司会とか SDC の取材とか平行しながらの３日間だったので、さすがに今日は起きるのつらかったです。

個人的には University Day にて登壇した James Gosling が現役大学院生からの質問に答えるセッションが楽しかったです。この内容については、SDC のニュースレターでも取り上げる予定なので、参加しなかった方はチェックしてください。



楽屋裏話
