Genunix.org

如果你想要最新版本的 OpenSolairs 以及其他基於 OpenSolaris 所建立的 distribution，你可以前往 Genunix 的網站下載。

在那個網站中你可以發現 OpenSolaris 更新的 build 版本，例如 Build 98, Build 99 等等，如果你的 OpenSolaris 已經裝好了，而你想要更新你的系統到最新的 Build 版本，你可以執行下列指令:

pfexec pkg image-update

假如你目前的作業環境不是最新的，這個指令將會把你目前的作業環境作 snapshot，然後更新你的作業環境並且將新的作業環境作 snapshot，接著更新系統的 Boot Environment 加入新的作業環境，使得你可以於開機程式中選取這個作業環境來開機; 這樣做的好處是透過 'pkg image-update' 不會動到你原本的作業環境，而且當新的作業環境有問題時，你隨時都可以回復到舊的作業環境。
