The below are links to Cryptographic Framework information.  This encompasses documentation the evolution of the framework from the design to Solaris 10, to OpenSolaris.  For the most part all the data accurate for OpenSolaris, but there may be some cases, in particular Solaris Encryption Kit, that is Solaris 10 only.

  • Solaris Security for Developers Guide:
    • Chapter 8: Intro to the Cryptographic Framework
    • Chapter 9: Writing User-Level Cryptographic Applications and Providers
  • Cryptographic Framework - Solaris Admin Guide
  • Kernel Crypto API & SPI man pages
  • Kernel Crypto Funcational Specification
  • Cryptographic Service Provider Interface
  • Walk through sample uses of kCF

Other information can be found in the Sun BluePrints

*Using the Cryptographic Accelerator of the UltraSPARC T1 Processor:  Providing info on how to use ncp with OpenSSL, Sun Web Server, Kernel SSL, and others via the Cryptographic Framework
