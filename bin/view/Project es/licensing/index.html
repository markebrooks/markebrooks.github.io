Licencias

Licencias

El código fuente, los ejecutables y la documentación puede ser descargada desde opensolaris.org. Este sitio, además, contiene vínculos a información que puede ser descargada desde otros servidores. La mayor parte de estos datos están bajo licencias que deben ser repasadas antes de ejecutar la descarga.

Esta página contempla los tipos mas comunes de licencias usadas en opensolaris.org. Toda la información de la licencia debe ser repasada cuidadosamente antes de la descarga.

Licencias del código

La mayor parte del codigo de OpenSolaris es difundido bajo la licencia Common Development and Distribution License (CDDL) Version 1.0 (.txt). La CDDL es una licencia Open Source Initiative (OSI)-approved. Por favor, revise la CDDL FAQ para más información sobre la CDDL.

El proyecto OpenSolaris además incluye una série de componentes basados en proyectos open source existentes. Estos componentes deben continuar siendo accesibles bajo sus licencias actuales. Por ejemplo,

  • Perl en la consolidación ON está licenciado bajo la licencia  The Perl Artistic License;
  • GNOME en la consolidación JDS  está licenciado bajo la licencia GNU General Public License;
  • X11 en la consolidacion X  está licenciado bajo la licencia MIT license.

Todo el texto de las licencias OSI-approved puede ser encontrado en el OSI website.

Cada consolidación/proyecto sobre opensolaris.org define la licencia que deberá ser usada para las contribuciones. Para más detalles, revisar la información individual del proyecto.

Licencias de los binarios

La OpenSolaris Binary License es la licencia usada para binarios construidos desde código fuente propietario o desde código fuente cubierto por multiples licencias. Algunos binarios se proporcionan así por conveniencia, mientras que otros binarios representan código que no puede, o todavía no está preparado, para ser código abierto. Para estos binarios, cada tarball disponible para descarga, contiene un fichero con todo el texto de la licencia Binaria de OpenSolaris. Para más detalles sobre este tipo de licencia puede visitar la OpenSolaris Binary License FAQ

Es posible que se encuentren disponibles para descarga otros binerios protegidos bajo otras licencias. Por ejemplo, puede haber drivers binarios bajo licencia de tipo BSD-style, y puede haber binarios construidos por conveniencia bajo código licenciado usando la CDDL (con lo que los binarios estarían protegidos también bajo la CDDL)

Licencias de la documentación

Los fuentes de la documentación de OpenSolaris están protegidos bajo una de las siguientes licencias:

  • La CDDL, usada para licenciar los fuentes de la documentacion que se entrega como parte de la estructura de empaquetado del software. Los ejemplos incluyen las páginas del man y ayuda en línea.
  • La Public Documentation License (PDL) v1.01, usada para las fuentes de la documentación que se entrega por separado del software que documenta. Los ejemplos incluyen los manuales técnicos tradicionales. Para más detalles sobre este tipo de licencia, puede visitar la PDL FAQ

Cuando sea apropiado, las licencias de terceros y las licencias open source se conservarán y se incluirán como parte de las fuentes de la documentación.
