LDoms Manager

Introduction to LDoms

Logical Domains, or LDoms, technology allows a user to allocate a system's various resources, such as CPU, memory, and IO into logical groupings or domains. This enables one to create multiple discrete systems, each with their own operating system, resources, and identity within a single physical computer system.
See the LDoms Community pages for more details.

The LDoms software stack is made up of 3 components

  • Solaris/OpenSolaris OS built-in drivers/services.
  • Platform firmware
  • LDoms Manager software

This project has been created to facilitate the ongoing development of the LDoms Manager software.

Getting the Source
