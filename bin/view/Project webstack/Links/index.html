Blastwave.org

The Blastwave commmunity has built some of their Web Stack components based on the best practices learned from Cool Stack. You can find out more about how to obtain software packages from Blastwave.org here.

Solaris Freeware Companion Software

The Solaris companion software includes a number of Web Stack components, along with other freeware, and is shipped with Solaris 10 and available for Solaris Express Developer Edition. It's also available as a download from that site. See either the Sun web page or the OpenSolaris.org Companion Project site for more details.

Currently, some components of the companion software are currently in the ARC process for update in Solaris Express Developer Edition. These packages evolve more slowly than many of the other packaging projects, therefore you should look to one of the others if you want the very latest releases.

Sun Freeware

For 14 years, sunfreeware.com has been providing packaged software for Solaris and now OpenSolaris. They also redistribute the Cool Stack packages. You can find details at sunfreeware.com.
