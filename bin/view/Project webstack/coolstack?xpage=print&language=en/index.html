Cool Stack Info

Important Note

The CoolStack project has been superseded by the Sun Web Stack product.

You can still download CoolStack as noted below for historical reasons. Please note the repositories mentioned below are no longer being maintained. Download Sun Web Stack binaries or sources if you want to get the most current version.

Cool Stack Project

 Cool Stack is a collection of some of the most commonly used open source applications optimized for the Sun Solaris OS platform. By using these binaries you will enjoy the best levels of performance from your system, while also reducing your time-to-service.

Cool Stack had been originally conceived to provide a set of out-of-the-box optimized binaries for a common set of software components on the UltraSPARC T1 based systems. By performing this packaging for the community, the OpenSPARC project and Sun's Performance Technologies group had a goal to make it easy for users to quickly add packages to their existing systems to quickly obtain optimized performance and reducing time to service.

Over time, there was sufficient demand for an equivalent set of packages on x64, so a similar set of optimized packages and build scripts were put together for the i386 and amd64 architectures as appropriate.

Cool Stack derived its name from the CoolTools project it is associated with.

 More Cool Stack information here: http://cooltools.sunsource.net/coolstack/

Please note that Cool Stack is not WebStack. Cool Stack predated and inspired the WebStack project but it is a separate effort. As noted above, Cool Stack builds stand-alone packages which can be installed onto Solaris 10. Cool Stack does not deliver components into OpenSolaris. For historical reasons, Cool Stack sources (see below) continue to be hosted here on the Web Stack project. For WebStack information, refer to the main project pages.

Obtaining Cool Stack Packages

Cool Stack is available for download from the Sun Download Center. Packages are available both for SPARC and x86 platforms. Source snapshot packages are also available for each version.

  • Final 1.3 release: Download - http://cooltools.sunsource.net/coolstack#Downloads
  • Documentation - http://cooltools.sunsource.net/coolstack#Documentation 
  • Change log - http://cooltools.sunsource.net/coolstack/changelog1.3.txt
  •  Cool Stack software Forum - http://forum.java.sun.com/forum.jspa?forumID=857
  • Our blog entry announcing the release - http://www.opensparc.net/blogs/2008-06/shanti-subramanyam-cool-stack-1.3-is-here.html 
  • Latest Release - Cool Stack 1.2 supports Solaris 10 and Solaris Express Developer Edition.
  • Previous Release - Cool Stack 1.1 supports Solaris Express Developer Edition and Solaris 10.

Latest Cool Stack Sources

The current source tree for Cool Stack is available from the webstack project svn repository. You can browse the Cool Stack source using the opensolaris.org source browser.

You can check out the source tree via svn:

% svn co svn+ssh://<username>@svn.opensolaris.org/svn/webstack/webstack

where <username> is your opensolaris.org user name. In order to have access to check out the sources you need to set up an ssh key in your user profile (even if you only want to do a checkout).

Please remember that this source repository is for the stand-alone Cool Stack builds only. If you are interested in the OpenSolaris WebStack project source, see here instead.
