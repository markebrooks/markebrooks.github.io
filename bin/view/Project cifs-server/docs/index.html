Documentation

Solaris CIFS Service Documentation

The OpenSolaris CIFS service project has formal documentation and documentation that can be created and updated by the community. The formal documentation, the Solaris CIFS Administration Guide and the man pages, is published at docs.sun.com. The community documentation is available from http://www.genunix.org.

You can participate in the OpenSolaris Documentation Community.

Presentations

Alan Wright delivers a presentation at SNIA SDC 2008 about the native OpenSolaris CIFS service.

Slides

  • Solaris CIFS Service (PDF)
  • SNIA SDC 2008: Native OpenSolaris CIFS Service (PDF)

Books and HOWTOs

The content of the following documents applies to Solaris Express Community Edition (SXCE) Build 90:

  • The Solaris CIFS Administration Guide.

The OpenSolaris CIFS Service wiki includes the following:

  • Troubleshooting information for CIFS-related topics, including the Solaris CIFS service, the Solaris CIFS client, and the identity mapping service.
  • A getting started document, which includes HOWTOs and tips for installing and configuring the Solaris CIFS service.
  • The what's new with Solaris CIFS document includes the list of features and feature changes that have been introduced in SXCE since Build 79.

This troubleshooting, getting started, and what's new information is available for update by the community at http://www.genunix.org.

Man Pages

The Solaris CIFS service man pages listed below are linked to the current Solaris Express Reference Manual Collection. You can view the man pages online by installing at least Solaris Express Community Edition Build 79. To have access to the man pages, you must install the SUNWman package.

The following man pages are new for the Solaris CIFS service:

  • smbadm(1M)
  • smbd(1M)
  • smbstat(1M)
  • smb(4)
  • smbautohome(4)
  • pam_smb_passwd(5)

The following man pages have been updated for the Solaris CIFS service:

  • chmod(1)
  • compress(1)
  • cp(1)
  • cpio(1)
  • ls(1)
  • pack(1)
  • pax(1)
  • tar(1)
  • sharectl(1M)
  • sharemgr(1M)
  • zfs(1M)

The OpenSolaris CIFS Service project uses the OpenSolaris wiki at http://www.genunix.org to include documentation that can be augmented and created by the community.

At this time, the following documents are available:
