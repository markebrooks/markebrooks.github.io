Wireless Networking

Wireless Networking for OpenSolaris

Solaris WiFi team is working on adding support for some of the most popular wireless network cards to Solaris.  All of the drivers and tools produced by the team will be pre-released to the OpenSolaris community and then integrated into Solaris.

Please send your comments and questions to <laptop-discuss@opensolaris.org> discussion forum.
- The WiFi driver compatibility list consolidated all the WiFi support for Solaris.
- Wireless Support FAQ

The following drivers and tools are currently available:

  • wificonfig - Wireless Configuration Tool
      - v0.2 was integrated into OpenSolaris build 29, v0.3 with WPA support is not integrated
  • ath - Wireless Driver for the Atheros Chipset
      - WPA-PSK support was integrated into OpenSolaris build 64.
  • inetmenu - Network Configuration GUI Tool
  • ipw - Wireless Driver for the Intel Pro/Wireless 2100 Chipset
      - WiFi GLDv3 version was integrated into OpenSolaris build 61, Solaris 10 Updates user may use v0.3
  • iwi - Wireless Driver for the Intel Pro/Wireless 2200BG/2915ABG Chipsets    
      - WiFi GLDv3 version was integrated into OpenSolaris build 61, Solaris 10 Updates user may use v0.3
  • cardbus - New Cardbus Driver With Support For 32-bit PC Cards
      - Integrated into OpenSolaris build 45
  • ndis - NDIS Wrapper Toolkit
      - Support Broadcom BCM43xx Chipsets
      - GLDv3 based ndis-1.1 was released to OpenSolairs.
  • pcwl - Wireless Driver for Agere 11b and PrismII 11b Chipsets
      - WiFi GLDv3 version was integrated into OpenSolaris build 60, Solaris 10 Updates user can use v0.1
  • pcan - Wireless Driver for Cisco Aironet 340/350
      - WiFi GLDv3 version was integrated into OpenSolaris build 60, Solaris 10 Updates user can use v0.1
  • ral - Wireless Driver for RaLink RT2500 802.11b/g Chipset
      - Integrated into OpenSolaris build 69.
  • rtw - Wireless Driver for Realtek 802.11b Chipset
      - Integrated into OpenSolaris build 70.
  • wpa_supplicant - WPA Supplicant
      - WPA/WPA2-PSK mode support was integrated into OpenSolaris  build 64.
  • wpi - Wireless Driver for Intel 3945 Chipset.
      - The GLDv3 version of the driver was integrated into OpenSolaris build 64
  • malo - Wireless Driver for Marvell Libertas 88W8335/88W8310 802.11b/g  Cardbus/PCI adapters.
  • zyd - Wireless Driver for ZyDAS ZD1211 802.11b/g  USB adapters. 
  • iwk - Wireless Driver for Intel 4965 Chipset.
      - Integrated into Opensolaris build 86
  • ural - Wireless Driver for Ralink RT2500USB 802.11b/g USB adapters.
      - Integrated into Opensolaris build 91
  • rum - Wireless Driver for Ralink RT2501USB/RT2601USB 802.11a/b/g USB adapters.
  • urtw - Wireless Driver for RealTek RTL8187L 802.11b/g USB adapters.
  • Sierra - Intel WiFi chipset support
      - iwk ph1.5 - iwk with Calibration and IBSS support
      - iwh driver - Intel WiFi Link 5100/5300 Driver(Shirley Peak)
      - iwk sparc version and iwh sparc version are ready to use, they depend on net80211 SPARC version.
  • rwn - Wireless Driver for Ralink 2700/2800 chipset.
  • atu - Wireless Driver for Atmel AT76C50x USB based 802.11b devices
  • arn - Wireless Network Driver for Atheros 9000 series chipsets
  • rwd - Wireless Driver for Ralink RT2561/RT2561S/RT2661 chipset.
  • uath - Wireless Driver for Atheros AR5523 USB chipset.
