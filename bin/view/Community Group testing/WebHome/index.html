Testing

  
  

The testing community provides a forum for discussing OpenSolaris test issues and works with developers to ensure quality. Input from the community is encouraged to help shape how we do testing and test development.

Testing Initiatives

OpenSolaris Test Links

 We have established pages listing useful OpenSolaris test links and performance benchmarks. If you know good ways to test OpenSolaris, send your ideas to the testing discussion list.

OpenSolaris Test Suites

 Test suites and benchmarks used to test Solaris are being open sourced so they can be used by the OpenSolaris community to test their code. Here's the link to the page that lists them.

OpenSolaris Test Projects

 Test developers are able to develop OpenSolaris test suites and start new OpenSolaris test projects. The Test Development Project is focused on test infrastructure development. Let us know if you want to start a new test project.

OpenSolaris Self-Service Testing

 A self-service functional and performance test application has been released that allows OpenSolaris contributors to submit test runs and review results. Here's the link to the self-service testing page.

OpenSolaris Test Results Repository

 A repository of OpenSolaris functional, regression and performance test results and baselines.

OpenSolaris Test Farm

 A test machine farm where OpenSolaris contributors can build and test their code on a range of supported hardware platforms. Here's the link to the test farm page.

OpenSolaris Unit Test Repository

 OpenSolaris developers will be able to submit unit tests as part of the integration process. These unit tests will be included in OpenSolaris test suites.

OpenSolaris Use Case Project

 The Testing Community is one of the sponsors of the OpenSolaris Use Case Project which captures OpenSolaris application and operating system use cases and real world user experiences.
