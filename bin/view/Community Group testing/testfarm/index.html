Test Farm

The OpenSolaris Test Farm allows OpenSolaris Contributors to build and test their code on various platforms prior to integration into the OpenSolaris code base. The Test Farm automated software is developed by the Sun Solaris Quality Engineering organization and the systems are maintained by the Solaris Labs organization. There are currently test farm sites in Broomfield, Colorado (BRM,USA) and Beijing (BJS,China).

Any OpenSolaris user can access test farm machines. More information on Access Requirements is given below. An automated build server and virtual machine interface is available that allows users to add accounts and zones. Test farm groups are used to control access to selected machines. Standalone test machines currently use manual reservations.

 The reservation interface is hosted at test.opensolaris.org/testfarm.

 The Test Farm usage is shown at test.opensolaris.org/testfarm/usage.jsp.

Access Requirements

 You will need the following to add accounts and reserve test farm machines:

  1. OpenSolaris User Name
     2. Bug or Project you are working on
     3. Accept the OpenSolaris Test Farm Terms and Conditions
