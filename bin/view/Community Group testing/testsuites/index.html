STC Test Gate

OHACTS Test Gate

 The Open HA Cluster Automated Test Suites perform functional testing of High-Availability Clusters. The test suites in this test gate use the Test Environment Toolkit (TET) framework with CTI for TET.

White Box Tests

White box tests are developed and maintained by OpenSolaris developers and are used for unit testing and regression testing of specific OpenSolaris features. These tests are included in the ON source tree along with the source code they test.
